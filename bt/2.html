宝塔windows面板介绍

宝塔windows面板介绍：

1、版本特性
基于Python重构的全新Windows面板，更轻更快，更加安全稳定，内存占用更少
新版Windows面板，已全部采用Python重构开发,安装包缩小近75%，仅4M大小。同时面板启动速度相比上一代版本，提升加载速度接近40%。对CPU和内存的资源占用也大幅度降低。

一键创建网站、FTP、数据库；
安全管理，文件管理，PHP多版本共存及切换；
新版本只有web端，取消了PC端。

2、系统支持
操作系统：仅支持Windows Server 2008/2012/2016/2019，64位系统，且未安装其它环境（5.4老版本支持Windows Server 2003 x86）建议选用64位系统。
环境需求：Microsoft .NET Framework 2.0 及以上
内存要求：内存要求最低1G，推荐1G以上内存。

3、环境支持
环境程序的主要有，IIS、Apache、Nginx、PHP、Mysql、SQL Sever、FTP、流量监控插件。

web服务器：IIS、Apache2.4、Nginx-1.11.13；
PHP版本：5.2、5.3、5.4、5.5、5.6、5.7、7.0、7.1；
数据库：MySQL5.5、MySQL5.6、MySQL5.7、Mariadb-10.1.16、SQL-Server2005、SQL-Server2008；
FTP：FileZilla Server软件；
