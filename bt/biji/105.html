宝塔BT面板新增了安全入口校验，这是一种安全机制，在登录地址的后面加上8位字符的随机码，如果宝塔后台登录地址的随机码不对就会提示“安全入口校验失败”，下面由宝塔面板教程栏目给大家分享解决方法：

另外，很多同学遇到使用阿里云ECS云服务器安装宝塔后无法登录的情况，这是由于安全组的问题，参考官方文档（添加安全组规则开发8888端口- 阿里云）安全组问题几乎是每个宝塔用户都会遇到的问题，建议参考官方文档。

链接：/d/file/p/20230325/25471.html class="contentsignin">登录后复制

安全入口校验失败

报错提示：

请使用正确的入口登录面板

错误原因：当前宝塔新安装的已经开启了安全入口登录，新装机器都会随机一个8位字符的安全入口名称，亦可以在面板设置处修改，如您没记录或不记得了，可以使用以下方式解决

解决方法：在SSH终端输入以下一种命令来解决

1.查看面板入口：/etc/init.d/bt default

2.关闭安全入口：rm -f /www/server/panel/data/admin_path.pl

原因分析：

安全校验是一种安全机制，在原登录地址：http://服务器IP/8888/后面加上8位字符随机码，改造后的登录入口格式如：http://服务器IP/8888/0f5602ui，如果不加随机码则无法登录。

解决方法

如何解决？很简单，如果您好记得这8位随机字符，那么拼接到登录地址后面即可；如果您忘记登录地址后面的8位随机字符，怎么办？有两种解决方法：

方法一：执行命令找回随机字符

ssh登录到你的服务器上，然后执行命令：/etc/init.d/bt default即可查看到带有随机码的登录地址、用户名和密码，如下：

[root@lamplnmp ~]# /etc/init.d/bt default
==================================================================
BT-Panel default info!
==================================================================
Bt-Panel-URL: http://123.123.123.123:8888/0f5602ui
username: root
password: 89d89afsk
登录后复制

如上所示，可以看到Bt-Panel-URL、username和password，那么直接登录吧。

方法二：关闭安全入口校验功能

嫌麻烦？那么可以直接关闭安全入口校验功能，以后直接使用http://服务器ip/8888的方式登录

ssh登录到你的服务器，执行命令：bt，如下所示：

[root@lamplnmp ~]# bt
===============宝塔面板命令行==================
(01) 重启面板服务           (08) 改面板端口
(02) 停止面板服务           (09) 清除面板缓存
(03) 启动面板服务           (10) 清除登录限制
(04) 重载面板服务           (11) 取消入口限制
(05) 修改面板密码           (12) 取消域名绑定限制
(06) 修改面板用户名         (13) 取消IP访问限制
(07) 强制修改MySQL密码      (14) 查看面板默认信息
(00) 取消                   (15) 清理系统垃圾
===============================================
请输入命令编号：11
登录后复制

命令编号选择(11) 取消入口限制即可取消。

LAMPLNMP建议使用第一种方法，安全入口校验是一种安全机制，还是开启的比较好，不然容易被他人猜到登录地址。
