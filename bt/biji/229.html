下面由宝塔教程栏目给大家介绍宝塔面板之Linux web 环境搭建，希望对需要的朋友有所帮助！

一、引言

我们购买服务器一般都是用来搭建网站的，但服务器又与虚拟主机空间不同，虚拟主机空间买回来之后是可以直接使用的，但服务器本身是没有web环境的，所以需要我们手动进行配置安装。搭建web环境的方法有很多很多，可以选择一键安装，或者是每个软件独立安装、使用面板等。这里就给大家介绍一种最简单的安装方法——宝塔面板

二、服务器准备

购买服务器供我们选择的服务商太多太多，比如：国内著名的有：阿里云、腾讯云、百度云。国外有：GoDaddy 、krypt 、SoftLayer等。在服务器系统上可供选择二点有Linux & Windows两种。个人比较推荐使用Linux系统。因为Linux系统的服务器价格相对便宜，占用资源也比较少，而且宝塔面板对Linux也是比较友好的。

三、宝塔面板安装

1.连接服务器

远程连接Linux服务器可以使用XShell、Putty。个人推荐Putty（免费）。

软件下载连接：
Putty ：https://www.putty.org/
Xshell： https://xshell.en.softonic.com/

在服务器系统中本次教程使用的是：CentOS / 7.2 x86_64 (64bit)

打开Putty之后在Host Name(or IP address)下方输入IP地址

Port下方输入端口号，SSH默认为22 ，然后点击Open连接

连接是弹出对话框点击“是”即可

输入用户名，密码登录，一般用户名为root

安装脚本：

yum install -y wget && wget -O install.sh /d/file/p/20230325/install_6.0.sh && bash install.sh
登录后复制

打开宝塔面板官网，找到安装命令粘贴进去执行即可。

安装成功后页面

面板安装完成后开始安装web环境

宝塔面板，是自带WEB管理界面的，所以操作是非常方便

我们使用安装完成后给出的宝塔面板地址进行登录 http://180.76.138.187:8888/7be4bca6

登陆后选择LNMP的编译安装

然后耐心等待即可，一般需要30分钟到60分钟，具体速度根据服务器性能决定。

LNMP环境安装的过程中是不能创建网站的，等到所有软件安装完成后就可以创建网站啦~
