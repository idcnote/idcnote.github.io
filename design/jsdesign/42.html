即时设计教程 • 图层样式管理

在选中某个图层或画板后，其所有的样式属性会显示在右侧属性面板中。

本篇内容介绍的是如何添加/删除样式项。

图层样式的类型

样式包括外观、背景色、填充色、描边、阴影。

 

样式的相关操作

除「外观」和「模糊」外，其他的样式类型都支持「添加（多个）样式」和「删除（多个）样式」。

添加样式

选中图层后，在相应属性栏点击「添加」按钮。

 

删除样式

删除单个填充项

拖拽一个样式项并将其拖出右侧属性面板，即可将该填充删除。

 

删除多个填充项

取消勾选样式项，点击「删除」按钮，可以删除该样式类型下所有未被勾选的样式项。

 

应用、取消样式效果

添加后的样式属性在勾选情况下对选中内容生效，取消勾选后该属性不生效。


 

复制、粘贴样式

选中图层后，右键图层菜单并点击「复制样式」，或使用快捷键⌘＋⌥＋C，可复制该图层的样式便于复用到其他图层上。复制的图层样式属性包括「外观」、「填充」、「描边」、「阴影」。 复制后，选中需要复用样式的图层，右键选择「粘贴样式」或使用快捷键⌘＋⌥＋V，可粘贴图层样式。支持同时选中多个图层粘贴以复用。


 

复制 CSS

选中图层后，右键图层菜单并点击「复制 CSS」，可获取该图层的 CSS 样式代码。复制后的代码可直接复用到开发环境，开发效率更高。
