百度竞价『否词管理大师』是什么？否词管理大师有什么用？

为了缓解“否词不够用”的痛点，也为了帮助广告优化师从更全局的视角、用更清晰的规则来使用否词，配合oCPC转化自动出价等其他能力，提高推广计划的整体操作、管理和优化效率。对于百度竞价否定关键词设置规则进行了全面升级，『否词管理大师』。

否词管理大师作为一款全局性的否词优化工具，将长期为客户账户下的否词提供全方位的清理优化建议，在帮助您释放否词储存空间的同时，移除有高商业价值等类型否词以提升转化。

注意：
1）部分账户因”否词设置不符合新规则”无法自动进入升级后的否词管理方案。
(否词新规则要求：一星账户任一计划下否词不能超过2000个，二星账户任一计划下否词不能超过4000个，三星账户任一计划下否词不能超过5000个)。
2）旧否词管理方案拟于2023年10月进行全面下线，全部账户将切换为新否词管理方案。

否词管理大师工具主要作用：

1、快速清理冗余否词，帮助账户 “一键升级”享受新否词规则红利：否词管理大师将帮助您自动监测计划下重复的否词，自动清理冗余否词、完成新否词管理方案升级。升级后单计划下可添加的否词数量将进行大幅提升 （一星账户最多可添加2000个，二星账户最多可添加4000个，三星账户最多可添加5000个）

2、提供全账户否词一体化管理能力：通过否词管理大师工具，您可以查看账户下所有计划里建议清理的否词类型，包括：计划与单元重复、全单元重复、与关键词冲突、低搜索量的冷门词、导致转化量受限共计5类否词。您可以通过否词管理大师工具选择性清理各计划下的否词，以清理出更多的否词空间并且移除会对您转化有影响的否词，提升管理优化效率。

 

否词管理大师产品入口：

  1. 推广管理 -> 定向 -> 关键词 -> 否定关键词 -> 前往否词管理大师
  2. 推广管理 -> 进入计划详情 -> 关键词 -> 否定关键词 -> 前往否词管理大师
  3. 推广管理 -> 进入单元详情 -> 关键词 -> 否定关键词 -> 前往否词管理大师
  4. 营销服务 -> 服务市场 -> 营销工具 -> 否词管理大师
