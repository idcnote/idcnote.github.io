文章正文

6月1日开始，微软向Windows 7/8.1用户推送免费预订Windows 10的广告，相信大部分的人都已经收到这条广告，除了少部分使用OS X系统的用户之外。

如果用户预定了Windows 10，那么在7月29日之前新系统将会自动下载到电脑中。而到了正式发布日当天，系统就会自动进行安装。

也许有人已经预定了Windows 10的升级，但有些人错过了。那么他们还有机会升级到Windows 10吗？

答案是肯定的，毕竟微软还希望扩大Windows 10的市场份额。微软表示，即使用户没有预定仍然可以升级到Windows 10，因为除了预定之外还有一个“Get Windows 10”应用来管理系统升级。其实，即使你卸载了Get Windows 10这个应用，你依然可以通过Windows Update获取全新的Windows 10操作系统。

另外，微软还表示，Windows 10发布后有些用户可能并不会立即受到推送消息，可能还需要等上几个星期。

Windows 10将在今年7月29日正式发布，如果你已经预定了，那么在系统发布当天将会自动安装。不然就只有通过手动下载系统安装了。

可是，微软免费提供Windows 10升级，天下真有这么美的事情吗？《福布斯》科技专栏作家戈登·凯利撰文称，Windows 10 并非真正免费，免费升级存在种种限制。免费策略可以帮助微软解决诸多问题，微软要让免费策略落到实处。

微软所谓的免费，其实是有条件限制的。你需要使用两款特定的系统，同时还要使用特定系统的特定版本，而且要在特定时间内完成升级。

从表面来看，一切都很行得通。两款Windows系统就是Windows 7和Windows 8，版本是最流行的家庭版和专业版，用户有一整年的时间来进行升级。数百万的用户都将会这样做，那么Windows 10将会成为史上用户升级速度最快的系统。

另外，戈登·凯利还分析称，这是微软对抗谷歌（免费软件+大量广告）、苹果（免费软件+昂贵硬件）和Linux（免费软件+无数发行版）的方式。在硬件广度、价格区间以及软件和服务的数量方面，微软远胜竞争对手。他建议微软，如果要向竞争对手发起全面反击，你必须让Windows 10真正免费。

他表示，Windows 10的普通升级业务对微软Windows营收贡献并不多，大部分还是来自企业用户和新PC买家。Windows 10免费升级服务不面向企业用户，而新PC买家看似免费使用Windows 10，但其实已经被包含在电脑的整机售价中。

另外，他还表示，在普通Windows用户中，最具代表性的是最精通技术、最具号召力和最具影响力的用户。他们可以帮助好友处理技术问题，提供购买建议，而且在传播正面或负面口碑方面发挥重要作用。因此，微软应该为他们真正免费提供 Windows 10系统。
