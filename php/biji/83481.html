备份数据大概是每个人都知道的建议了，但大部分人都不见得会这么做。预测数据丢失是一件几乎不可能的事情，但许多企业至今仍然没有可靠的备份系统。IT工作人员现在也可以轻松管理Azure的备份，确保所有的数据都安全地被存储在了云端。通过赋予企业在任何时间安排他们数据备份的时间弹性，可以避免上载高峰，网络也不至于会被堵塞。现在你只要下载Azure Backup agent软件就可以开始了。

Windows 10客户端设备现在已经可以通过“Azure 备份”服务无缝备份至云端。使用64位 Windows 10 系统的用户能够以一种安全的方式将他们设备中的重要文件夹数据备份至 Azure 中加以保护，之后也可以在任何设备中恢复数据。下面是“Azure 备份”服务的功能介绍：

教程：

1.将设备中的单个文件与/或文件夹备份至 Azure。

2.自定义备份时间安排，一天可设置3次自动备份时段。

3.通过转移增量备份副本（即只从之前结束的备份点开始转移变更数据）高效利用带宽。

4.即便数据已上传至云端也可以进行加密。

5.你可以让备份副本在 Azure 中保存长达99年时间。

6.在 Azure 中轻松浏览备份的文件与文件夹，随意恢复需要的文件。

7.将存储在 Azure 中的备份副本无缝恢复至其他任何设备中。

8.对上述所有操作（如规划备份次数）进行监控，用户可在一个熟悉的微软管理控制台界面完成数据备份与恢复操作，如下图所示。

9.“Azure 备份”服务现已覆盖全球17个国家与地区，您可以选择最近的地区来备份数据。
