文章正文

微软已经宣布Windows 10将拥有7个不同的版本，而部分用户在选择时可能并不确定哪个版本才是最适合自己的。为了帮助有意升级的用户提前做好选择，科技网站Digital Trends日前就撰文介绍了Windows 10各个版本之间的区别和最适合的用户群体。

Windows 10家庭版

Windows 10家庭版面向的是所有普通用户，它向笔记本、台式机和变形设备提供了Windows 10的所有基本功能，可满足个人家庭用户的使用需要。

作为Internet Explorer的继任者，全新的Edge浏览器将会负责所有的网页浏览任务，新的Continuum功能可根据变形设备当前的工作模式自动对用户界面进行适配。除了照片、邮件、地图、日历、音乐和视频等基本应用之外，Windows 10家庭版甚至还加入了与Xbox One的整合，可让用户将家用机游戏串流到PC上进行控制。

总结：这个版本适合大多数的PC用户

Windows 10移动版

Windows 10移动版的职责是替代现有的Windows Phone品牌，它承诺将解决WP在应用资源上的匮乏，让微软的移动平台在实力上更加接近iOS和Android。

Windows 10移动版当中所预装Office办公套件据称可以提供“几乎无妥协的体验”，当中所包含的功能和使用方式都是用户在PC上所熟悉的。

在使用搭载Windows 10移动版的设备时，用户将能够获得和桌面版操作系统几乎相同的体验，只不过一切都为移动设备进行了特殊的优化。从Outlook到照片应用，系统中的一切都会和桌面版Windows相整合，并加强了对于云服务的依赖性。

总结：这个版本是手机和平板电脑的选择

Windows 10高级版

Windows 10高级版在家庭版的基础上又新增了一些额外的功能，官方称可“满足小企业的需求”。和家庭版一样，高级版同样是为PC、笔记本和变形设备所准备的。

Windows 10高级版所具备的Windows Update for Business功能可让企业更加效率地对软件升级进行管理，Secure Boot和Device Guard等工具还提供了更多的应用和设备管理功能。此外，该版本还提供了CYOD（自选设备办公）所需的云技术支持。

总结：PC爱好者和小企业的首选

Windows 10企业版

这一版本所具备的功能在Windows10高级版的基础上再一次进行了拓展，它所提供的高级功能可满足大企业的需要，帮助他们对抗日益增长的安全威胁。

和高级版一样，企业版也将具备Windows Update for Business功能，但又新增了一种名为Long Term Servicing Branches的服务，可让企业拒绝功能性升级而只获得安全相关的升级。

Windows 10企业版只会通过VOL渠道发布，普通消费者无直接购买渠道。

总结：适用于大企业和其他大型机构，普通消费者无法轻易购买

Windows 10教育版

Windows 10的版本虽多，但当中的一部分并不是面向公众的，比如教育版。

和企业版一样，Windows 10教育版同样只会通过VOL渠道发布，学校和学生可通过某种指定步骤从家庭版和高级版进行升级，具体方式会在近期公布。此外，微软还并未透露该版本会具备哪些特别的功能。

总结：和企业版之间的区别还并不十分明确，但它显然是学校和教育机构的首选

Windows 10移动企业版

和企业版类似，Windows 10移动企业版适用于工作环境当中的移动设备，其所具备的大多数功能则和标准的Windows 10移动版相同。

这个版本同样只能在VOL渠道购买，它的最明显目的是让移动设备在符合企业规范的前提下工作。企业的IT部门可以统一控制系统升级，安全和创新相关的功能可自动被添加，这不仅为用户提供了实时保护，同时避免了因为疏忽所导致的安全风险。

总结：适合想要为员工手机和平板推行标准化设备管理政策的组织使用

Windows 10物联网核心版

物联网核心版可以算作是最不同寻常和最有趣的Windows 10版本了，它所瞄准的设备类别包括智能恒温器、机器人、甚至是多士炉。

Windows 10物联网核心版的最大意义是加强通讯设备和家用电器之间的连接。比如说，在它的帮助下，我们的智能手机将可在需要时自动对烤箱进行预热。

不过物联网核心版的应用范围并不仅限于此。微软表示，无论是HoloLens、树莓派、Windows计算机还是手机，它们都是一个平台，你可以将它们统统拓展至物联网版本。

总结：物联网厂商和爱好者将通过这个版本来实验兼容Windows的小装置
