文章正文

在移动互联网吞噬一切之时，PC已经静如死水多年。不过今天PC老大哥微软发布Windows 10的玩法，则让PC市场又回到人们视野，以完全不同的方式：微软这一次彻底地拥抱了互联网。支持包括盗版Windows在内的设备免费升级，整合微软 Cortana、XBOX One游戏平台，屏蔽售卖授权码、走向免费+增值的互联网模式。

同时，Windows 10学习移动App分发方式与360、腾讯、联想和小米合作分发“操作系统”，并且对IoT（物联网）设备进行了全面支持。操作系统通过第三方进行分发可谓史无前例，微软正在学习互联网玩家，快速抢占用户扩大市场份额。值得注意的是，为什么选择的对象是这四家，而不是别人？

联想：

联想宣布将在其中国 2500 家客服中心和指定零售店面第一时间提供 Windows 10 升级服务。这将帮助中国用户将自己的新购或者现有 Windows 设备轻松便捷地升级至 Windows 10。此外，联想还宣布他们将致力于 Windows Phone 的开发，在2015年年中推出中国移动订制版 Windows Phone。

联想是世界第一大PC厂商，是中国第一大PC厂商，在PC市场的地位举足轻重，不论是存量还是新增市场，很大一部分Windows设备都在其手上，只不过现在是反过来了，不是联想向微软交授权费，而是联想帮助微软分发Windows。

联想在PC厂商中拥有最广的线下服务网点，这对于部分低端用户、下沉市场用户有很大的普及效果，要知道这里一部分人现在还在用XP只是他们不知道还有WIN7这些新系统。联想也宣布在中国 2500 家客服中心和指定零售店面第一时间提供 Windows 10 升级服务。

联想的“PC+”战略与微软的“设备+服务”战略不谋而合，联想PC第一，智能手机全球第三，同时联想还在积极布局平板、智能硬件市场。微软的Windows 10同样是面向多设备的OS，包括IoT设备。联想也表示要在2015年推出中国移动定制版Windows Phone设备。

360：

奇虎 360 宣布将帮助其中国用户无缝地免费升级到 Windows 10。奇虎 360 将帮助超过 5 亿Windows PC 用户仅通过加速下载的几次点击实现 Windows 10 升级。

360是软件免费模式的始作俑者。免费模式让360占领了大量的Windows设备，现在中国是恶意软件感染率最低的国家，360从安全已延展出浏览器、网址导航、搜索引擎等业务，微软现在的免费模式本质上与其一样。

360用户海量用户基础，公开数据表明其PC端的活跃安全用户为5.09亿，覆盖率超过了96.1%。它已是与腾讯比肩而立的中国两大桌面客户端厂商。海量用户基础让它可以帮助微软快速分发Windows 10，360已推出傻瓜式升级工具。

360过去帮助Windows修修补补，用户升级补丁基本是用360，360还协助微软修复漏洞多次获后者致谢。因此它在安全上的优势使之成为Windows 10独家安全合作伙伴。

4、本次4家公司中只有360是老大出来站台，周鸿?出现在了深圳活动现场，说明360对这个合作还是很重视的。除了前面提到的两家确实有许多合作契机外，还有一个原因或许是因为360最近在做手机，与Windows还是存在很大的合作想象空间的，比如Windows PHONE手机。

腾讯：

腾讯宣布将为其中国 Windows 7 和Windows 8.1 用户提供 Windows 10 的免费升级，并将 QQ、腾讯视频、腾讯电脑管家加入 Windows 10 免费升级包中。腾讯还承诺将为其拥有超过 8 亿中国用户的旗舰应用 QQ 开发 Windows 10 通用应用。腾讯广受欢迎的《英雄联盟》、《穿越火线》、《地下城与勇士》等 PC 游戏也将支持 Windows 10 和新的 Windows 商店。

腾讯是中国最大的客户端厂商之一，同时是最大的移动互联网公司，旗下8款App进入TOP10名单。从用户基础和用户属性上来看是非常好的操作系统分发平台；

腾讯同样是免费+增值模式的集大成者，只不过是从IM切入。腾讯会把 QQ、腾讯视频、腾讯电脑管家加入 Windows 10 免费升级包中，《英雄联盟》、《穿越火线》、《地下城与勇士》等 PC 游戏将支持 Windows 10 和登录 Windows 商店。预装第三方内容让Windows 10学习手机厂商玩起软件预装，成为软件分发商。

Apple Watch首先支持微信，说明腾讯在西方科技巨头、在科技世界中话语权已经足够强大。微软不论是要在PC端重拾话语权，还是在移动端找到自己的位置，都要选择与这样的巨头结盟。

小米：

小米宣布布与微软展开一项全新的合作。在今年晚些时候，部分小米 4 的忠实用户将受邀帮助微软测试 Windows 10， 并为其未来发展贡献意见。这些用户将有机会下载 Windows 10 技术预览版，并向微软反馈其使用体验。

小米与Windows 10的合作还比较浅，只是粉丝优先公测的方式。毕竟一个是移动端的新贵，一个是PC端的失势的贵族。不过小米正在成为中国最大的移动设备厂商，在手机之外的智能硬件上不断攻城略地，已推出路由器、智能手环、小米盒子、小米电视、智能摄像头、运动摄像头、空气净化器诸多产品，智能手机是Windows一直想要进入，而智能设备则是Windows“设备+服务”战略下“设备部分”，Windows 10已支持部分IoT设备。所以微软选择小米，是认可它在智能手机市场的地位，也是看好它在未来IoT世界的潜力。“设备+服务”的转型让微软必须重视小米这样的新兴巨头，先与之搞好关系。

Windows 10全新的发布方式，自然会帮助Windows最新版系统快速普及到海量PC设备，避免WIN8等版本难以普及的问题。通过免费进入设备之后，微软便与这些用户建立联系，不断加载更多自己的服务，现在选择中国的巨头、专家和新贵，是明智之举。免费模式在消费者软件业务上终于不再有死角，因为连最不可能免费的Windows都已沦陷。
