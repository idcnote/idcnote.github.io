Windows 10正式版已兼容大多数Windows7/8.1应用

文章正文

大概还有几周的时间，微软的全新一代操作系统 Windows 10 就要正式发布了，很多人开始议论究竟是否要升级 Windows 10 操作系统。除了关心 Windows 10 究竟有什么新鲜的功能，是否符合微软提出的一系列升级条件之外，更多的人已经开始琢磨着升级到最新的 Windows 10 系统之后，自己常用的软件和程序是否能够正常运行，尤其是工作用的电脑上会不会罢工，或出现意想不到的情况。

7 月 6 日消息，对此，微软认为正在使用 Windows 的用户完全可以放心。近日，微软 Windows 操作系统负责人特里·迈尔森（Terry Myerson）表示，绝大部分 Windows 7 和 Windows 8.1 上现有的应用程序都可以在 Windows 10 上正常运行。不过秉着负责任的态度，他也承认，当升级到新系统时，部分可能会有一些小问题。

如果真的发生兼容性的问题，用户其实会有多种选择，包括是否继续 Windows 10 的升级，或者取得与软件开发商的联系，让他们知道你遇到了需要解决的问题。

迈尔森说： “在我们数亿百万计的系统测试过程中，我们看到了绝大多数 Windows 8x 和 Windows 7x 上的应用程序已完全与 Windows 兼容，不过这样的工作永远不会停止，我们每一天都会继续为应用程序与设备的兼容性而努力，毕竟这是我们致力于打造 Windows 服务的一部分。”

别担心，7 月 29 日你还不一定可以升级，还会有先人帮测试

若个人计算机上存在应用程序兼容性问题，7 月 29 日将暂时无法获得 Windows 10 升级，迈尔森接着解释称，因为微软希望为客户提供升级服务时，个人的电脑已经是迎接新系统的最佳状态。不过，Windows 10 升级势不可挡，兼容性的问题迟早会得到解决，只不过升级到 Windows 10 可能要等待更长的时间。

迈尔森还表示，当然，即使存在兼容性的问题，升级应该也不会存在任何问题，用户照样还是可以完成升级。之前根据微软官方的描述，正常情况下 Windows 7 和 Windows 8 操作系统上的应用程序、文件和设置均会在升级之后保留到 Windows 10 上。不过，也有些应用程序或设置可能不会迁移，比如防病毒和反恶意软件应用程序，某些第三方应用程序或者功能设置也并不会出现在安装完成的 Windows 10 上。

因为，这涉及到驱动程序和固件支持、应用程序兼容性和功能支持等。微软官方也提供了相关建议，当不确定是否应用完全兼容的情况下，升级前最好通过 Windows 10 升级顾问来对 PC 或平板电脑进行个性化评估。

另外，微软也确认，Windows 10 正式发布之后，整个升级过程可能持续数天、数周，甚至数月。不过全部参与 Windows 会员计划的用户，将在 7 月 29 日获得正式版 Windows 10 升级。侧面的意思其实是，之前帮助测试的“小白鼠”用户，还是会在第一时间升级完成继续进行测试，帮助即便是 Windows 10 正式版的系统寻找问题，造福更多用户。换句话说，将会有一大帮人帮忙测试兼容性，纠结是否升级的完全不必担心。
