文章正文

对从7月29日起，Windows 7和Windows 8.1用户将能下载Windows 10。对于微软及其客户来说，这是一个重大转变。但微软提供了大量信息，以简化升级过程，让用户尽可能方便、简单地升级到Windows 10。然而一去三十载，软件帝国想要改变游戏规则：全新的商业模式、全新的产品发布战略、全新的升级控制，所有一切皆发生变化。然而所有一切又都汇集成一个问题，到底是否应该升级到Windows 10？

考虑到Windows 10上了一个大大的头条，「免费」可以说是非常容易理解的。但实际上并非如此，笔者自2014年10月起就开始使用windows10的第一个beta版本，然后笔者收到了高大上的软件安装包，费用就是来自这个存储媒介。

是否升级到Windows 10是一个非常个人的决定，完全取决于个人的需求和考量。就Windows 10操作系统带来的各种惊喜及种种不足，笔者会在下文中一一赘述并给出最终结论和升级建议。

好的一面，Windows 10是免费的

Windows 10的确是免费的，微软在7月17日也终于证实了这一点，产品官网显示Windows 10的主流支持将持续到2020年10月13日，延长支持会持续到2025年10月14日。这可是一个大好消息，这两个时间跨度与Windows早期付费版本的操作系统软件以及微软未来的「视窗即服务」的产品战略不谋而合，用户可安心使用Windows 10直至2025年延长支持服务终结。


图示：Windows 7和8用户迫不及待想要升级到Windows 10

需要注意的是，只有在Windows 10发布日之后的12个月内（2016年7月29日），用户才可以享受到免费升级的优惠政策，如果超出这个优惠时期范围，用户就得要花费119.99美元和199.99美元来购买 Windows 10家庭版和Windows 10专业版。

横跨移动桌面计算多个平台，Windows 10无处不在

以OS X操作系统核心为基础，创造出移动操作系统iOS的苹果公司倍感荣耀，而目前微软的举动似乎要更深一步，Windows 10可运行在台式机电脑，笔记本电脑、平板电脑以及智能手机，虽然用户界面会略有不同，但是依然采用的是同一核心代码。

可以说Windows 10是目前唯一一款横跨台式机电脑，笔记本电脑、平板电脑以及智能手机四种计算平台的操作系统产品。这也带来了极大的便利和优势，比如Continuum功能（类似于苹果产品上的Handoff功能），可根据用户的操作方式、系统环境以及屏幕大小来自动调整系统的自适应功能，无需手动切换。用户可在不同设备之间获得无缝对接的平滑用户体验。

目前微软正在全力追赶苹果的Handoff以及谷歌基于云的服务（比如 Docs, Sheets and Maps）并大有赶超态势。所有 Windows 10的应用都可以运行在任何Windows 10设备上。一个应用程序拥有桌面版和移动版等多种版本的尴尬将彻底终结。

需要指出的是遗留应用程序并不在Windows 10跨平台的支持列表内，用户只能在自己的手机中运行Office for Windows 10，而非 Office 2007。

更懂用户的语音助手Cortana

Windows 10最为特色的功能莫过于语音助手Cortana。

作为苹果Siri的强劲竞争对手，微软的Windows智能手机语音助手Cortana终于走进Windows 10。用户什么都可以问，Cortana 都能瞬间秒答。此外Cortana还能处理诸如打开浏览器、撰写新邮件、设置提醒或者日程安排等命令。

图示：Cortana的出现意味着微软在桌面计算平台上首次与苹果公司Siri展开正面对抗

和所有语音助手功能一样，Cortana远非完美。对于那些羞于与计算机对话的用户而言，更为通情达理的Cortana会让用户倍感舒心。

虚拟桌面

就技术层面上而言，虽说 Linux 是首个提供虚拟桌面功能的操作系统平台，但是人们所熟知的却是OS X平台下的「Spaces」多桌面切换功能，现如今微软的操作系统也玩起了多桌面切换功能。Windows 10操作系统用户可以创建多个虚拟桌面，一个用来工作、一个用来休闲娱乐，另一个用来制定假日计划。

图为：Windows 10虚拟桌面功能演示

支持DirectX 12并整合 Xbox 游戏平台

对于游戏玩家而言，DirectX 12也许是升级Windows 10的唯一一个重要原因。尽管早期有报道称，相较 DirectX 11，DirectX 12会带来30-40%的性能提升，但实证表明也有接近10%-20%的性能增幅。悉数历史上每款全新DirectX所带来的图形性能的提升，可以说DX12是PC游戏发展的未来。

Windows 10兴许是Xbox One游戏机的未来，玩家可通过Windows 10将Xbox One上的游戏投射到PC和笔记本电脑上，尽管目前阶段，受限于家庭Wi-Fi的连接速率，投射后的游戏画面有些延迟，但实际上已有相当不错的可玩性，整个的设置过程也非常简单。

从某种程度上来看，笔者认为，微软整合Xbox游戏平台的策略可将一些Windows 10用户从PS4游戏主机平台吸引到微软的Xbox One游戏平台上（本文不会在此阐述次观点）。

图示：用户可通过Windows 10将Xbox One游戏投射到电脑上

超低的硬件配置要求

虽说Windows 7操作系统发布也有六年之久，但是Windows 10对硬件的要求并无显著提升。官方推荐的最低硬件配置如下所示：

处理器：主频为1 GHz或更高的处理器或SoC芯片；

内存：32位系统至少需要1GB内存、64位系统至少需要2GB内存；

硬盘空间：32位系统至少需要16 GB硬盘存储空间、64位系统至少需要20 GB硬盘存储空间；

显卡：支持DirectX 9或者更高版本并带有WDDM 1.0驱动的独立显卡；

显示器分辨率：1024×600。

简而言之，如果你手头上有能免费升级到 Windows 10 的计算机（Windows 7 或者 Windows 8.1），那么玩转 Windows 10 根本不成问题。

Windows 10的优点

就微软对Windows 10那种半遮半掩的低调做法，笔者深感不爽。尽管一些关键问题仍未得到官方答复，但是Windows 10操作系统的核心真的是非常棒。

Windows 10无论是看起来还是用起来都像极了Windows 7和Windows 8的混合体。Windows 10运行速度超快，安全性绝佳，开始菜单的回归也让习惯了Windows 7的用户（这些用户往往对Windows 8有排斥心理）赞不绝口。

图示：Windows 10操作系统巧妙地将新特性和广为熟知的过往功能有机结合起来

Snap分屏功能得以加强，仅需几次点击，用户即可根据需要调整各个分屏的大小；文件传输功能更加智能化，移动超多大容量的文件再也不会拖慢系统运行速度了；全新的多媒体播放器内置更为全面视频解码支持；通知中心能够在一个地方将微软和第三方应用程序的通知分开，当然还会有很多超赞的功能特性，限于文章篇幅，笔者不再一一列举。

笔者承认自己是一个喜新厌旧的升级狂魔，早期Beta版本的系统图标的确让人感到惊悚，而目前看来，图标更加精细和现代化了。虽说Windows 10优点不胜枚举，但是缺点也不能忽略。

Windows 10的缺点，并非所有用户都能免费升级Windows 10

尽管Windows升级仅仅是公司营业收入的很小一部分（微软绝大部分的营业收入来自新PC的销售），但令人恶心的是升 Windows 10还是有一系列限制的。

首先，Windows Vista和Windows XP的用户就被微软排除在免费升 Windows 10的优惠政策之外。考虑到Vista和 XP依旧占据了全球13%的PC市场份额，并且这些用户极有可能会选择升级操作系统，微软这种政策的确令人不解。

另外Windows 7企业版、Windows 8/8.1企业版以及Windows RT/RT 8.1用户也被微软排除在免费升级Windows 10的优惠政策之外。

强制性更新

Windows 10最为争议的就是用户自由更新系统的控制权。

Windows Update系统更新功能中的Nvidia显卡驱动更新与Nvidia官方的驱动产生冲突，导致部分用户产生画面显示故障、游戏崩溃等各种问题，严重了影响了系统的稳定性。

对于Windows 10家庭版（大部分用户是一般消费者）以及Windows 10专业版（大部分用户是电脑爱好者和商业用户）的用户群体而言，用户可选择接收、下载并安装微软通知下的任何更新。这些更新可能是重要的安全更新、功能更新（即便用户不想使用）以及驱动程序。

延迟更新是很有可能会发生的，Windows 10家庭版可延迟一个月，而专业版用户或许可选择延迟到8个月之后，过了延迟截止期限之后，这些系统更新还是会自动安装到用户电脑中。这其实是Windows 10 EULA（最终用户许可协议）的一部分，用户在使用并安装操作系统软件之前就已经「同意并接受」了这些条款，当然其后果也是非常麻烦的。

就在上周，一个不兼容的显卡驱动以及一个各种bug缠身的安全更新自动安装到数百万台Windows 10 beta测试用户的电脑中，最终致使用户电脑在没有安全警告的条件下重复崩溃。即便用户将这些麻烦的更新移除掉，Windows 10也会在后续自动安装这些问题多多的更新。回顾历史，微软的更新一直争议重重，仅在今年，微软就发布了40多个问题更新补丁。

令人困惑不解的是，微软发布了一款能够卸载Windows 10更新的工具，但是这款迟到的工具必须要提前安装才能移除不必要的麻烦更新，需要指出的是这款工具并非Windows Update系统更新功能的一部分，貌似违背了微软自己订立的EULA。

微软已经发布了KB3073930更新补丁，用户可以借此隐藏或屏蔽所有的Windows Update和驱动更新，有了这款工具，用户可以直接从列标勾选需要移除的系统更新和硬件驱动。笔者就此询问了微软，但后者拒绝就此发表评论。

Windows 10家庭版用户竟成小白鼠

强制性更新导致Windows 10家庭版用户成为无辜小白鼠，家庭版用户既不能像专业版用户那样延迟更新长达八个月，又无法像Windows 10企业版用户那样能选择停止所有更新。

微软在官方博客中写道，「等到商业用户的电脑都更新了，相应的变化会被数以百万计的业内人士、消费者或客户所验证。长达数月之久的内测，会让系统更新部署更有保证。」

微软就是这样赤裸裸地承认了，消费者被当做商业测试的小白鼠。也许Windows 10的免费升级不是白吃的午餐，对于终端用户而言还是要付出另外一种代价。

另一方面，现如今Windows盗版用户也可以注册成为Windows 10 beta版本测试用户，虽然可以获得完全免费的正版Windows操作系统软件，但是却没有任何延迟更新的选择，这就意味着这些用户很可能成为了微软不良更新的牺牲者。

图示：有关Windows 10免费升级的官方细节依然冗长乏味、扑朔迷离。

云里雾里的执行策略

毋庸置疑，微软强制自动更新的策略引众怒。笔者当然了解微软的初衷是好的，强制性更新升级确保每个用户的PC都能保持最更新，进而最大化确保系统安全和可靠性，但问题在于强制性遇到了用户的抵抗。

为了应对强制性升级，终端用户往往会通过暴力黑客方式来停止系统更新的自动安装，但是今年四月，微软产品市场经理海伦·哈米兹 (Helen Harmetz) 则表示，强制停止安装Windows 10更新包的用户可能会无法获得今后的系统安全更新包。

笔者无法从微软官方获得直接答复，微软最近发表声明，「客户可预计到，我们会在Windows 10更新策略上采取Windows 8.1更新的方式。」

也就是说，如果用户未能在30天内将Windows 8升级到Windows 8.1，这些用户将无法获得今后的安全更新包，对于商业用户而言，这一更新截止期限延迟到120天，鉴于Windows 8.1是一项庞大的系统更新。如果微软将30天不更新就切断安全更新包的做法应用到每个更新补丁上，想必会引来更多用户的不满和愤怒。


图示：用户在Windows 10再也看不到经典的Windows 7上的《蜘蛛纸牌》游戏

被微软移除的某些功能特性

尽管某些小的功能特性极有可能会影响到少量用户，但是微软还是决定移除掉某些Windows 8中的功能特性。最为明显的当属Windows Media Center功能，一些热爱此功能的用户已经在论坛上表达了自己的失望和不满。其他被微软移除的功能特性有DVD回放（实际上第三方影音播放软件解码器丰富，根本不是问题）、桌面小部件以及对 USB 软盘驱动器的支持。

各种Bug满天飞，Windows 10并未准备就绪

笔者认为，Windows 10给人的主要感受像是硬生生地火线推向市场，虽然不能将其比拟为「死亡蓝屏」那么严重，但各种小故障还是令人心烦意乱。

目前笔者在两台PC上发现任务栏上的图标莫名消失的情况，开始菜单也时不时拒绝响应，然后就是各种驱动不兼容的问题，想必第三方厂商还在为兼容Windows 10系统做准备。而现阶段，微软自行开发的全新浏览器Edge显然是一个壳而已，不支持任何插件功能并且标签功能太少。

与此同时，微软已承诺会在今年十月推出Windows 10的重大更新包（由于微软不再使用Service Packs字眼，因此这次更新可能会被成为『Windows 10.1』），此外微软在Windows 10正式发布前的两天内火线推送大量升级更新补丁。


图示：Windows 10虽然不错，但是仍需时日逐步完善

Windows 10更新：再等等也不迟

就微软操作系统发布的历史看来，全新的Windows发布的经验法则就是「再等等」，微软会利用这些时间赶快修复早期出现的重大Bug。笔者认为此规律在Windows 7上并未显现，Windows即便是早期的Beta版本也是非常扎实可靠的。但是历史的经验法则又重新回到 Windows 10上来。

在某种程度上，这种情况的确让笔者在内的科技宅们倍感失望。但从长期看来，笔者认为Windows 10会成为微软历史上史无前例的最棒的操作系统产品，可以肯定的是Windows 10是自Windows 95以来最为激动人心和最具野心的产品，并且还是免费的（尽管有条件），Windows 10仅仅是没有充分准备好而已。

先不谈Windows 10的各种优点，现阶段的各种Bug满天飞，在自动更新以及强制性政策依然未能获得官方肯定的答复，尤其是目前的自动更新所导致各种严重的系统稳定性问题，但笔者认为现实还是要付出一些代价的。

所以希望大家再等等，也许微软粉丝以及前沿科技宅会无视我的建议，毕竟他们已经做好了为尝试新特性而不得不面对各种缺陷的心理准备。但就一般用户而言，鉴于免费升级Windows 10的期限长达一年，所以不要急着去升级Windows 10。

Windows 10终究是一个伟大的产品，用户最终还是要升级的，但是仍需耐心等待，至少要在微软在今年十月推出Windows 10.1重大更新包之后再来升级也不迟。
