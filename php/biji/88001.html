六、判断变量是否是真
 
console.assert()用来判断一个表达式或变量是否为真。如果结果为否，则在控制台输出一条相应信息，并且抛出一个异常。
 

复制代码 代码如下:

<script type="text/javascript">
　　var result = 1;
　　console.assert( result );
　　var year = 2014;
　　console.assert(year == 2018 );
</script>

1是非0值，是真；而第二个判断是假，在控制台显示错误信息
