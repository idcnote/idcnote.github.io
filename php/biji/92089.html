　Osx 10.10.3怎么样？Osx 10.10.3正式版到底好不好用呢？Yosemite此前的两个升级版10.10.1、10.10.2都是以Bug修复、可靠性改进为主，而这次的10.10.3则首次为新系统引入了不少新功能，尤其是新的照片应用Photos for OS X，将取代Aperture、iPhoto。

　　就像之前预测的那样，苹果发布了 OS X Yosemite 10.10.3 ，这也是 OS X 系统更新至10.10之后第一个重大更新。OS X 10.10.3 今年2月开始测试。OS X 10.10.3 可以通过 Mac App Store 完成更新。之前的 Yosemite 更新补丁，包括 OS X 10.10.1 和 OS X 10.10.2 都修正了 Bug 并改进了稳定性。

　　体验表明，Photos for OS X是基本合格的，速度很快，编辑工具更全面、强大，但也缺少很多高级功能，比如插件、放大镜、可调整笔刷、自定义元数据域等。

　　同时新系统还将emoji表情整合到了单独一个可滚动的页面上，并加上了分类标签，而且对表情库进行了扩充，甚至可以调整肤色(免得去告人家歧视)，并升级了iPhone、iMac、Apple Watch的表情。在系统设置中配置Google账号的时候，两步验证也得到了支持，因此不再需要针对应用的单独密码。Force Touch API也开放给了开发者，可以在自己的应用里加入力度感应手势了。

　　除了照片应用，OS X 10.10.3 还包含了全新 emoji 选择器，新选择器将所有种类的 emoji 集成在同一个窗口下，并且新增了其他肤色表情、额外的国旗 emoji 以及 iPhone、iMac 和 Apple Watch 设备 emoji。下面是详细更新内容。Photos 照片 应用采用 Yosemite 风格设计元素，大量使用透明和扁平元素，并可以与 iCloud 照片图库深度集成。

扁平化界面与半透明风格设计：

　　意料之中的，这次全新的 OS X Yosemite 和 iOS 一样全面采用了扁平化的设计风格，重新绘制了应用的图标，随处可见的半透明磨砂玻璃元素效果，整体视觉感受与 iOS 基本保持一致，至此，苹果的扁平化终于一桶浆糊了。

HandOff - 革命性的跨设备体验：

　　以往的桌面系统和移动系统都是独立的个体，它们之间难以沟通。如果你在 iPhone 撰写邮件写到一半，你希望回到电脑前继续编辑，那么就得考虑如何将那段文字发送到电脑上。之前我们介绍过的 PushBullet 之类的工具可以稍微方便地帮助用户做到这一点。

跨设备的电话与短信：

　　依然是围绕 “OS X 和 iOS 连通” 这个主题，如今如果你的 iPhone 有来电，那么你可以直接在 Mac 或 iPad 上接听，也能在它们上面拨打电话。同样，短信方面也和上面的 HandOff 特性一样，你可以直接在 Mac 或 iPad 上发送、接收、回复、查看全部的短信，完全不需要去拿起你的 iPhone。

　　另外，OS X Yosemite 已经支持发送类似微信的录音短信了，此外，也可以发送视频短信。而且，现在即便是网页上的电话号码，你也可以通过 Mac 直接拨打而不需再翻包包去找手机了，懒人福音啊有木有。

iCloud Drive 客户端：

　　之前苹果的 iCloud 服务一直是整合在各种 App 里面的，它可以为 App 的设置、文档等提供同步功能，但我们是不能像 Dropbox 那样将它当成一个网盘来使用的。iCloud Drive 终于可以让 iCloud 有了「网盘」的样子，我们可以存储任何文件在它里面，可以随意管理、创建文件夹等，它会自动同步，客户端支持 Mac、iOS 以及 Windows。终于，苹果让 iCloud 有了它原本应该有的样子了！

　AirDrop 跨平台传送文件

　　为了配合 HandOff 更加方便的跨平台连通性，AirDrop 不再只能在 iOS 间传送文件了，这次 OS X Yosemite 上的 AirDrop 终于支持万众期待的 OS X 和 iOS 间的文件传送了！

Spotlight - 智能的全局搜索工具：

　　OS X Yosemite 将 Spotlight 工具进化到一个全新的高度。如今，Spotlight 拥有了一个全新的搜索框界面，你不仅可以使用 Spotlight 来快速启动APP，还可以搜索本地文件、联系人、日历、邮件、信息、词典、地图、系统设置等内容，而且现在还能搜索到 App Store 和 iTunes Store 上的全部应用和音乐电影等内容。当然，它还可以搜索互联网(维基百科、Bing等等)。Spotlight 在这次得到了大大的加强，看来类似的第三方工具 Alfred 等 App 有危险了。

MailDrop、Markup - 邮件的进化

　　为了解决在邮件中发送大型附件（如高清视频）的问题，苹果发布了 MailDrop，当你发送一个大容量附件后，这个附件会保存到云端服务器，然后 MailDrop 会生成一个安全的访问链接，邮件接收者可以通过这个链接访问和下载附件，工作原理和 QQ 邮箱的超大附件中转类似，看来这些小特性还是国人想得比较早做得比较快嘛。

　　而 Markup 则是允许用户在邮件中添加图片附件之后，对图片进行涂鸦注释再发送。比较有意思的是，它会自动识别你画出来的元素，譬如箭头或者泡泡会自动变成一个精美的图案。这是一个贴心的功能，苹果将它做得高大上了。

Safari 浏览器

　　Safari 浏览器不仅在外观上大变了，拥有了新的收藏页、标签页管理器，而且现在也支持 WebGL、SPDY、IndexedDB、JavaScript Promises 等众多新特性。新标准的加入，大大降低了 Safari 产生的功耗，官方表示可以让同一台 MacBook Air 通过 Safari 播放 Netflix 的 1080P 高清流媒体视频节省 2 小时的电力！

通知中心

　　和 iOS 一样，这次 OS X Yosemite 的通知中心也增加了“今日”标签页，用户可以在“今日”中查看日历、事项提醒、天气以及股市等等信息。不过，这次苹果还开放了API，可以让第三方的APP也能将信息显示到今日标签页中去。

Instant Hotspot - 超方便地在 Mac 上使用手机共享的移动网络热点上网

　　iPhone 一直以来都提供了热点共享功能，可以将手机的移动网络共享给电脑或其他设备上网。之前，你要先在手机上启用热点共享，然后再回到电脑上连接热点的WiFi。而现在 Instant Hotspot 可以让你更懒，即便你的 iPhone 丢在包包里面，在没有开启热点共享的情况下，你也能直接在 Mac 的 WiFi 信号中找到你的 iPhone 共享热点，你可以随时连接上去，并且无需输入密码，安全又方便。并且当使用过后，它也会智能地断开连接并不会浪费iPhone的电量。
