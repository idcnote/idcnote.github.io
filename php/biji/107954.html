文章正文

相信作为 Windows 用户，你应该很容易区分 Windows 7、Windows 8/8.1 和 Windows 10 这三个操作系统，尤其从界面设计来区别，比如是否有开始菜单或开始屏幕，或者是否可以从 Windows Store 商店下载应用程序等等。但是从根本上来说，同样重要的安全性、性能和移动性之间的变化你很难区别，随着 Windows 新版的发布，你可能知道哪些功能移除了，但是操作系统底层显著的变化，你却很难感受得到。

话说，今天 Windows 10 就要发布了，你对 Windows 7 和 Windows 8 的印象又是什么呢？

Windows 7 ：更快、更小、更好

Windows 7 诞生于 2008 年。在 Windows 7 中你能明显感受到用户界面在功能上的变化，那就是任务栏。全新的任务栏支持你将图标和快捷方式固定在上面，同时属性中加入显示最近使用项目的跳转列表，而最明显的变化是任务栏中出现了 Aero Snap 特效。另外，微软也进行了操作系统底层的大量改进，安全性和运行效率显著提升。

Windows 7 发布之时微软公布了新的最低硬件要求时，你必须要更强大硬件配置的电脑才能更好运行。这一点完全可以理解，毕竟这也是配合摩尔定律的发展，即集成电路上可容纳的元器件的数目，约每隔 18-24 个月便会增加一倍，性能也将提升一倍，所花的钱能买到的硬件和电脑性能，每隔 18-24 个月也翻一倍以上。

所以，我们看到了上网本的大量普及，虽然有一定的性能影响，但移动计算变得越来越普遍了。而且，很多功能如 Wi-Fi 无线网络、蓝牙连接、移动宽带和睡眠/休眠等功能变得越来越实用了，微软增强了这部分功能的稳定性，考到移动性微软甚至还在 Windows 7 中加入了对触摸屏和加速度计的支持。

同时，Windows 7 为了改善移动性的电池续航时间，微软也加入了电源管理功能，支持 BISO 或硬件驱动程序管理电源，未使用中的设备系统可以自动关闭更多不必要的东西，根据当前负载情况和性能要求来动态的调节 CPU 性能，比如降低 CPU 运行频率，减少所谓的空闲资源的能耗，自适应的显示亮度调节等，当没有任何任务执行时系统只有很低维持运行的功耗。另外，Windows 支持中断应用程序与 CPU 的通信，只有被一些事件触发后才会重新连接。

Windows 7 开发过程相当艰辛，这是微软首次花了五年时间来打造最精简、最高效的 Windows 内核，也就是一个名为“MinWin”的最小化内核项目。它被看作是微软后生之年的救世主，因为该项目能够帮助 Windows 成功瘦身，使之变得更容易驾驭，此后微软可以更轻松的改进系统架构，单独为系统添加新的功能，而不必重写核心服务，也不必担心会破坏上层 API 和核心服务之间的依存关系。

Windows 8.1：改进不仅只有磁贴

全屏幕的开始屏幕、超级按钮以及移动设备全屏化 Metro 风格的磁贴应用，这些是 Windows 8 最明显添加的全新功能，并且大肆强调触摸屏功能，完全疏远大量的传统桌面用户。有意思的是，微软推出了一个几乎纯 Metro 元素的基于 ARM 处理器的 Windows RT 系统版本。

但是，Windows 8 实际上还有很多不是那么明显的升级，尤其是增加了对新硬件的支持，显著改进安全性能，比如加入了UEFI Secure Boot(安全启动)，改良生物识别安全功能，防火墙也得到了更新和加强， Windows To Go 选项支持直接从 U 盘运行 Windows，改进 DirectX 性能 ，Hyper-V 虚拟机升级到与 Server 2012 相同的版本等等。

另外，微软也改善了电源管理模式，引入 Connected Standby 连接待命（待机状态）模式，类似于现在的智能手机中的锁屏待机状态。即当系统进入休眠状态时，应用程式虽处于暂停的状态，但依旧会与网路维持连线。该模式在移动设备中意义重大，保证电子邮件不会遗漏或延迟，实时资讯能够时刻保持最新状态。Windows 8 还能监测每一台 PC 中每一个组件的功率使用情况，然后选择以最有效的方式提高电池续航。

Windows 8 发布一年之后，首次重大升级 Windows 8.1 发布了，该升级重新带回了开始按钮，直接启动到桌面，对 Metro 界面进行了一些改动，但也加入了一系列安全和管理功能，为企业用户添加更多新硬件的支持，比如利用 Miracast 将屏幕内容串流共享给另一台设备，通过 Wi-Fi 直接打印。

Windows 8/8.1 开始深度集成 OneDrive（之前称之为 Skydrive），你的 Microsoft 账户登录任意 PC 之后，不只是系统设置和商店里的应用程序会同步，你保存在 OneDrive 云端的文件也会伴随同步。

Windows 10：既是前进也是后退

随着 Windows 10 的发布，微软针对跟随 Windows 8 而来的 Metro 应用程序有了新的想法，将其改造并命名为通用应用程序 （Universal apps），这种应用程序不再只是全屏运行，而是可以像传统应用程序那样在桌面上运行，而且支持跨设备运行，包括 PC、Windows 10 手机、平板电脑、Xbox 和 Hololens 等，支持根据用户设备尺寸，自动调整界面大小适应各种屏幕。

OneDrive 与 Windows 10 整合更加紧密了，现在你还可以选择哪些文件夹需要同步。之前在 Windows 8 上出现的超级按钮消失了，转而带来的是移动设备上早已流行的操作中心，用于显示新的通知，以及一些快捷操作方式。Windows 10 还带来了一个全新的浏览器，称之为 Edge，而 Internet Explorer 将不再是默认浏览器，未来也只能存活在企业设备上。

同时，Windows 10 也带来了很多安全方面的改进，尤其更加完善统称 Windows Hello 的生物识别技术的安全功能 ，支持指纹、虹膜检测和刷脸识别等各种认证方式。Windows 10 的性能得到了不断改进，现在新系统安装更快更简易了，而且安装完成之后还能自动清理，腾出更多的磁盘空间。

Windows 10 的用户界面进行过了重新设计，其目的是为了保证 Windows 10 既适合传统的键鼠操作，也能够在平板电脑等触摸屏设备上轻松使用，很多界面的改进都是基于 Windows 7 和 Windows 8.1 的混合体，比如回归之后的开始菜单。

另外，Windows 变成了一种服务，你可以通过持续不断接收新的升级而得到新的功能，而不必重新安装一个重大更新的 Windows 版本，也就是说 Windows 10 将是最后的 Windows 版本。更重要的是，微软希望你一直在 PC 上使用 Windows，因为新升级和功能会自动推送到系统上，你不用再纠结是否要更换另一个版本的桌面平台了。
