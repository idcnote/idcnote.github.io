Windows 10试用半年用户心得 老用户请放心升级但却不会太爽

文章正文

在期待今晚的微软 Windows 10 发布会的同时，让我们来回顾一下到目前为止的 Windows 10 相关消息。尽管我半年前就开始了 Windows 10 的测试过程，但还有很多人等待这次里程碑式的发布，就让我为这些还在等待的人来做一个点评。

1、版本号问题

首先，有一些媒体提到了为何版本号直接到 10 而不是 9。这跟 Windows Vista 到 7 不一样，因为 Vista 的版本号正是 6。BBC 认为 ，这是为了跳过人们对 Windows 8 比较失败的联想，并且让人和 Mac OS X 联系起来，进行直接对比。

而 新浪科技的编辑 似乎得到了独家消息：因为有一些软件在判断操作系统版本的时候，遇到“Windows 9”会判定为 95 或者 98，以兼容性角度出发，才选择了 10。

虽然很让人好奇如果到 Windows 20 的时候是不是也会跳过去，但是好像我们已经没有这个机会了。因为，Windows 10 有可能是从版本号来说最后一个大版本 Windows，在此之后，将不断的推出具有微小功能改进的新版本，就像苹果的 MAC 系统一样，它的大版本号始终停留在 OS X。不如说，Windows 10 也将会最后演变成一个商标。

2、开始菜单

Windows 8 设置了一个非常陡峭的学习曲线，老用户会感觉到不太习惯。其实即使是从事 IT 相关新闻报道的我们自己，也会一开始找不到关机按钮在哪里。好在，开始菜单在 Windows 10 回到了我们眼前。

Windows 8 是一个很明确的想要为带触摸的平板电脑设计的系统。虽然微软最早提出了人们现在正在使用的平板电脑的原型 Tablet PC，但是最后做成功的却不是它。在 Windows 现在面临的市场竞争环境当中，尽管它依然牢牢占据在 PC 领域的老大，但 PC 这种形式都不再是人们访问互联网和完成工作的主要形式。所以在那时，一个只是为平板而设计的系统就这样出现了。

其实，就在前几天，我趁着 8.1 仍然在活跃的最后几天机会，到微软和联想的旗舰店去体验了 Windows 8.1 的触屏机器。其结果是，哪怕是在触屏上，我依然不是用的特别习惯，因为平常在 iPad 上的几个多任务手势都无法实现。当然会有人说分屏比较好用，但是我更喜欢 4 个手指滑动，直接从一个应用转换到另外一个。而且如果有人疼惜那个 home 键，他现在也有了除了小白点之外的选择，就是用几个手指做抓挠的动作，撮一下屏幕，就会回到主屏幕。

相比之下，8.1 的手势需要在手指移动到屏幕上某个位置的同时，眼睛要盯着屏幕。比如说呼出多任务切换界面，是从屏幕的左侧边缘向右滑动一点点，再弹回去。这个“一点点”到什么程度？你眼睛必须看着屏幕。相比之下，苹果的几个手势操作是非常自然和顺畅的，即使只摸到了屏幕，也可以进行类似操作。也许这就是在触屏时代的“盲打”。

所以在经历了如此之多的问题之后，Windows 8 已经成为了一个不管什么人都不会太好评的系统，成为 Windows Vista 一样的滑铁卢，也就是一个历史的定论了。The Verge 的主笔沃伦写道 ：

“微软曾经有先把事情搞砸，再在下一个版本当中把我们拯救回来的传统。Windows 7 拯救了 Vista 用户……Windows XP 拯救了 Me 用户……现在 10 就是来把我们从 8.1 的噩梦当中拯救出来的。”

所以开始菜单就这么回来了，但愿真的能够拯救我们。

3、然而依然是触屏优先

在 Windows 10 升级的过程当中，自然绝大多数需要升级的机器都是 PC。尽管如此，微软依旧将 Windows 10 设置为一个触屏优先的系统。针对触摸屏所设置的，那些原本存在于 Windows8 和 8.1 上的功能，并不会因为讨好 PC 用户而消失。正相反的是，微软尽一切努力把原先不得不依赖 x86 系统的功能，都向着能够兼容平板和手机的统一应用（Universal Apps）方向迁移。

首先就是 Windows 附件。Windows8 在这件事情上完全做反了，它提供了一个激进的用户界面，但是对于附件的迭代却反而小心谨慎，它里面拥有两套计算器，两个录音机，两个日历，两个邮件。用户可以选择卸载其中一个，而他们一般选择卸载的，往往是微软想要大力扶植的统一应用版本；因为首先是用不习惯，其次是在 8.x 系统当中，这类应用的打开和运行效率极其低下，根本无法同 x86 应用相提并论。

在最新的用于正式发布的 Windows 10（版本号 10240）当中，现在已经可以非常顺畅的打开统一应用了，以至于我使用内部自带的 Groove 音乐程序，替代了原先的 QQ 影音，成为默认播放音频文件的选择（不过请原谅，视频播放器我还是用不习惯）。自带的看图软件，跟上几个版本的图片查看器相比，加入了图片美化的功能，这些功能之前只有通过下载 Windows Essentials 增强包才能做到。

跟上几代系统相比，邮件、日历和人脉 3 个应用得到了界面升级，使用起来也更加赏心悦目。不过很显然 ZDNet 的编辑 并不喜欢这个改变，他希望能够保留传统的收件箱，而不是都汇总成会话模式。他说：“在微软提供这个功能选项之前，我将永远不会使用自带的邮件客户端。”（顺带一提，无法访问 Gmail 的用户，可以使用这个自带邮件客户端，毫无压力的添加 Hotmail 和国内邮箱。）

在这一切的铺垫下，大量 x86 附件被砍掉。希望下次能继续砍掉剩下的几个，比如画图（可被替换成 Fresh Paint）和写字板（可被替换成 OneNote）。统一应用的推广，正如我之前所说的那样， 促进了微软对生态圈的开发和控制 ，应用商店也会越来越有用。开发统一应用比 x86 难度明显降低，还可以方便移植到 iOS 和 Android，希望有更多开发者能加入进来吧。

其次，一笔带过 Cortana，因为哪怕我是用 FaceTime 效果拔群的 MacBook Air，同样小娜会经常听不清我说什么。那你想象一下 PC 参差不齐的麦克风设置，恐怕你只能打字来提问吧？

4、Edge 仍然是个好人

这点真的不想作过多解释。尽管有编辑很客气的说，现在暂时不使用 Edge 只是因为没有扩展，但我相信有了扩展他也不会去用的。从小到大，微软浏览器给人的作用就是为了下载别的浏览器。

具体到这一次 Edge 的浏览器体验，如果之前曾经用过针对平板设计的 ie 浏览器的话应该大致知道是怎么回事。如果手里有支笔，现在可以在上面圈圈点点，并且发送到 OneNote。然而……

我这次去西单大悦城的微软授权经销店，店员还在给我推 Surface 的笔，还有 OneNote。说到圈点了我就说一句，我也许真的不习惯用笔写了。1）精细绘图我可以放大画幅用手指操作；2）大批量文字输入我情愿用语音。所以下次营销 Surface 请给我推荐一点圈点之外的东西。Edge 也是一样。圈点居然是 Edge 至今为止唯一的卖点，真让人心疼。

5、结论

开始菜单回归和操作习惯向 7 靠拢，体现的是讨好用户；坚持发展统一应用，坚持触屏优先，这是坚持原则。Windows 10 面临这两项不同方向的目标，结果它是怎么做的？用胶水把两块粘到一起。

就是这种感觉。甚至图标都没统一。在传统桌面和涉及到统一应用的部分，可以说 Windows 10 只是很不情愿的给了一个替代方案，在能保证让 Windows Insider 参与者心服口服的前提下，做了最小程度的妥协。

之前我提过，Windows 10 完全放开了全球用户提意见，大家还可以顶某个具体建议，还能当论坛跟微软一起 high。然而这也就是保证产品能卖出去，最后还是要以干掉 x86 这个变态目标作为最高奋斗纲领。而在论坛里普通用户居多，他们希望的是什么？也许是用一辈子 Windows 7 或者甚至 XP。

我觉得我可以跟着微软一起向这一批用户保证：你们从 Windows 7 迁移到 10，除了界面将就一下忍忍（不可能有 Aero 了），其他操作习惯完全 OK！放心大胆的升级就好了。

但是，世界是你们的，也是微软的，但终究会是我们完全不懂的那帮孙子的。他们会在电子产品的浸泡中长大，生下来就了解程序员的隐喻，甚至小学会学习编程。当他们再回头看 Windows 8 那样的重大变革，他们会怎么说呢？

“这帮老古董，不就是开始菜单没了么，连关机都不会，真笨！”

——大概是这样吧。
