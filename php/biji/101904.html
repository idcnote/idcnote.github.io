文章正文

本周，微软在总部的发布会上分享了更多有关 Windows 10 操作系统的信息。Windows 10 实际上标志着微软一个全新的改善，不仅大量 改进了用户界面，而且大量的新功能也加入其中。不过，除了系统本身所有伟大的新功能之外，微软似乎还暗示了些许未来 Windows 的方向。

在主题演讲结束之后，微软 Windows 部门高管 Terry Myerson（特里·梅尔森）表示，Windows 10 实际上可能是最后的独立的 Windows 版本，因为微软可能将更换不同的策略，将更多重心转移到系统的大更新上。

“大更新”取代独立 Windows 的消息过去已不止一两条

实际上，此前已经有不少相关信息。早在去年 8 月，著名微软观察专家 Mary Jo Foley 声称，下一代 Windows 将可能是微软最后一个独立的重量级 Windows 版本了，微软将加快 Windows 的更新节奏，更愿意选择经常发布大的更新或改进，以及补丁的方式通过 Windows Update 推送。

随后到了去年年底，微软新 CEO Satya Nadella（萨帝亚·纳德拉）也谈到了旧 Windows 时代的做法已经过时，认为过去 Windows 改进和升级一直相当缓慢。他还透露称，Windows 10 将改变以往的做法，重点在于推送给用户的重大更新节奏将发生改变，使其具备“脱胎换骨、更新换代”的味道。

Satya Nadella 表示，以往微软采用的都是每三年向客户发布新一代操作系统的做法，但现在与过去所做的完全不同，不是“这三年来”Windows 做了些什么改进，而是“这段时间”。

只以 Windows 为核心

Terry Myerson 强调称，从现在起，电脑运行的就是 Windows，而不是特定发布版本的 Windows，“版本”这个概念将不再用于操作系统。他还表示，这种变化将为操作系统带来额外的价值，而 Windows Update 则是升级和更新的主要渠道，不仅提供安全更新和补丁，而且还提供新的功能和性能改进。

“Windows 将作为一个服务，就像任何互联网服务，关于‘你在用什么版本呢？’的问题都将变得没有任何意义。这对我们 Windows 开发者来说是好消息，同时 Windows 应用程序将实现跨设备通用，开发者开发一个 app 就能支持最广泛的 Windows 设备了，包括 PC、平板电脑、手机、Xbox、物联网设备等。”

Windows 就是服务，请开启 Windows Update

微软之所以称 Windows 为服务，是因为其对设备的生产方式的改进产生了积极的影响。正如微软所说，Windows 10 之后可能将不会再发布新版本的 Windows 系统，而是尽快做好重要的大更新，用户无需再用数月或数年的时间等待全新新功能的 Windows 版本，所有的带新功能的重要、重大版本的更新，均会立即地、自动地、直接地推送到系统，而不需要完全重装。

关于这一点，Terry Myerson 似乎用 Satya Nadella 的原话重新表达了一遍，不过也是最好的证实。

“Windows 10 体验的演变将发生变更，并且随着时间的推移会变得更好。当新功能准备完毕之后我们就会立即通过 Windows Update 推送，而不需要等待下一个主要的 Windows 版本。我们认为，在未来几年内 Windows 将能够成为这个星球上最大的互联网服务之一。”

他最后还表示，每一个用户都应该保持 Windows Update 开启，无论使用的是平板电脑、智能手机还是 PC，因为这是获得新功能改进的主要途径。

“使用 Windows 10 的用户，尤其是企业级用户，最佳的做法就是开启 Windows Update，我们会尽快为他们提供最新的安全和功能升级，提高生产效率。”

微软已经确认，Windows 推出后的一年时间里，Windows 7 和 8.1 老用户都可以免费升级，微软希望用户尽快升级到 Windows 10。不过，微软却一直没有具体说明 Windows 10 最终正式版的发布日期，只是简单的提到了今年下半年。
