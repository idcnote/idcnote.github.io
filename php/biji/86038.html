文章正文

        Windows10可以说是微软在发布前最受关注的操作系统，人们似乎都想看看在windows8、windows 8.1后，微软究竟能否给用户带来更多的惊喜。对于微软Windows 10操作系统也是关乎其成败的一款系统，Windows 8、Windows 8.1低迷市场，让Windows 10承担了更重的责任，这是微软只许成功，不许失败的操作系统。

       Windows 10一直吸引着广大用户的高度关注，各种关于Windows 10的消息就层出不穷。这个月末就是微软一年一度的开发者大会，据说会宣布下个Windows 10 Build有关消息。而且据透露，Windows 10的RTM 版很可能在6月份就要向OEM厂商供货了。对于关心Windows 10的网友来说，我们不妨通过最新的预览版来看一下Windows 10默认程序的真面貌，下面就让我们先睹为快吧。

开始菜单

1.开始菜单和开始屏幕从此同根生

       Windows 10中开始菜单和开始屏幕终于和大家见面了。从这一版本开始，开始菜单和开始屏幕不再单打独斗，而是从内容和形式上都达到了统一，区别只在于大小。

       默认是开始菜单，但是一键即可切换到全屏的开始屏幕，而且二者也有诸多共通之处，包括活动磁贴。这将吸引很多Windows老用户的的欢迎，也将促使更多的网友愿意采用这个新的操作系统。


Windows 10日历

2.Windows 10日历版本更精致

       在Windows 10 最新的技术预览版本中，我们发现日历应用的界面已经进行了升级，看起来与邮件应用更加一致。

       在延续Windows 8版本中的一些基本元素的同时，新版本显得更加精致一些。同时用户还可以在日历中设置一些小功能，方便用户使用。

Windows 10相机

3.Windows 10相机

       Windows 10不仅仅将用于PC产品，同时还将覆盖智能手机和平板电脑等移动平台。Camera应用程序也将成为Windows 10的默认原生相机应用。

       Camera将出现在PC和平板电脑上，而且用户甚至在三星或HTC未来推出的Windows 10设备上也将能享受Camera带来的优秀拍照和后期编辑体验。

邮件应用

4.Windows 10邮件应用

       Windows 10邮件应用的界面迎来了全新的设计，风格上与其它组件也更加统一。相信在Windows 10正式到来的时候，整个系统都会与Windows 8的对比都会更加鲜明。

       新版邮件应用在布局上没有太大的改变，左侧栏依然是邮件列表，排序(分类)选项在最左列，顶部还有回复选项。点击左上角的“汉堡”(三道杠)图标就可以唤出“新邮件”面板。

地图

5.Windows 10地图

       地图是目前很多人都会用到的一个功能，在Windows 10中的地图应用，包括街道和卫星视图、本地搜索数据以及语音导航等，它还将微软必应地图和诺基亚HERE地图的最好功能有机整合起来。

       在地图上，你可以搜索企业获得其详细资料，包括所在位置、联系信息、照片以及来自大众点评网站Yelp的评价等。你还可以利用这种应用预定餐厅等。如果你外出旅行，这款地图应用可以提示你吃饭和购物场所，当然还有当地著名景点以及各种活动等。

播放器

6.Windows播放器

       Windows播放器的功能在之前的版本中已经出现，而在Windows 10中，Windows播放器添加了更多功能。

       你可以使用这款应用浏览、管理并播放你设备中的所有内容，还包括OneDrive或添加至Music Pass中的音乐，作为一位Music Pass用户，你也将了解并爱上‘探索’与‘收音机’功能。

应用商店

7.Windows 10应用商店

       Windows应用商店是最新版本。就目前而言，在当前的Windows商店应用的主要区别是，它的布局被设计为通过垂直滚动。所以它更适合浏览台式机或笔记本电脑鼠标上。

Video

8.Video

       新的视频应用程序可从Windows应用商店测试版单独下载，Video支持浏览和播放视频，支持MKV支持，浏览播放电影电视剧。接下来也会支持从Store内购买电影电视，离线播放，改进搜索和电影评价等。

录音

9.录音

       在Windows 10中，Windows 8/8.1录音机已更名为录音。但它仍然具有相同的功能，但它的UI现在有一个浅色的主题（如修订后的报警及时钟和计算器应用程序）。

Photos

10.Windows Photos

       在Windows 10中，你可以通过Photos查看存储在本地设备或OneDrive账户上的图像，并且可以进行简单的修改和修饰。

       总结：上面就是笔者为大家列举的Windows 10所带来的一些变化，这还远远不是Windows 10的诸多优势，如果有兴趣的用户，不妨下载最新的预览版来亲身体验下，只有亲身体验后，才能感叹Windwos 10的伟达，值得你选择。
