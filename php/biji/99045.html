对于几十年来依靠销售Windows创收的微软来说，对操作系统免费其实是断臂自救，以有限的收入损失来换取在移动互联网生态大战中绝地反击的机会。微软最终还是向中国用户妥协。在昨天举行的Windows硬件工程产业创新峰会上，微软公司操作系统事业部执行副总裁特里·梅尔森宣布，Windows 10将于今夏在全球190个国家和地区以111种语言版本正式发布。

梅尔森同时宣布了与联想、腾讯和奇虎360等多家大中华区全球化公司建立一系列战略合作及计划，让数亿Windows 中国用户免费升级至Windows 10，这标志着Windows真正迈进免费时代。

Windows 10升级

盗版升级后仍然是盗版

Windows 10是微软最新一代的全平台操作系统，将统一PC、手机、平板、Xbox等平台，与以往的版本不同，Windows 10将是一种服务，而非单纯的产品。Windows 7、Windows 8.1、Windows Phone 8.1用户都可以在Windows 10上市的第一年内免费升级。微软希望在Windows 10推出后，可以在最短的时间内让更多的用户体验和使用。

特里·梅尔森表示，中国现在是全球创新中心，向世界各地出口Windows PC，中国的技术社区和用户对于Windows 10的未来至关重要。为此，微软通过中国的合作伙伴，将让数以亿计的中国用户能够轻松免费升级到Windows 10，“我们想通过Windows 10与用户紧密连接起来。”

至于盗版软件能否实现免费升级，微软官方表示，任何符合要求的设备都可以免费升级到Windows 10，包括盗版Windows，“相信随着时间的推移，用户会认识到正版Window的价值，我们将为他们提供更轻松便捷的方式转换为合法版本。”

根据GFK提供的数据，使用正版Windows的用户在不断增长。从技术角度而言，Windows 10可以在非正版Windows的设备上实现升级；但从软件授权角度而言，其Windows不是正版的，盗版Windows软件升级到Windows 10之后仍然是盗版的。

微软官方进一步解释称，如果用户本身电脑用的是盗版软件，原来在电脑里面的病毒、恶意软件等都会对软件和电脑产生影响，因为你升级之后的Windows 10仍然不是正版。

业内人士评价，对于几十年来依靠销售Windows创收的微软来说，对操作系统免费其实是断臂自救，以有限的收入损失来换取在移动互联网生态大战中绝地反击的机会。

Windows 10普及

微软将与国内IT巨头合作

中国是个人电脑普及率最高的国家之一，如何在最短时间内在用户中普及Windows 10？微软选择与各领域内市场份额占比较高的巨头进行合作，比如联想、腾讯、奇虎360。

峰会上，联想集团副总裁、中国区总经理童夫尧宣布，联想的全线消费类PC新品、Windows平板新品将在今年夏季统一预装Windows 10系统。此外，联想还将成为国内第一家参与微软TechBench项目的厂商，自Windows 10正式上市日起第一年内，为所有国内联想用户提供Windows 10系统升级服务。

奇虎360也在会上宣布与微软进行合作，通过无缝升级把Windows 10免费带给其中国用户，用户仅需加速下载的几次点击即可实现Windows 10升级。

此外，微软宣布与腾讯达成战略合作，面向广大中国用户提供定制版Windows10。该版本将内置新版QQ、腾讯电脑管家、腾讯视频等产品。随后，用户可以通过腾讯官方渠道免费升级到该版本的Windows 10。只要一年内升级，即可永久免费使用。用户只需确保PC在联网状态，无须光盘、密钥等外设，即可通过腾讯电脑管家一键升级。腾讯也将在其旗舰应用QQ上开发Windows 10通用应用。

北京青年报记者还了解到，通过与小米的合作项目，部分小米4手机的忠实用户将被邀请参与Windows 10体验，他们能够有机会下载Windows 10技术预览版——装载系统并向微软提供反馈。

Windows 10创新

支持用户刷脸或指纹等登录

在Windows 10平台上，硬件和设备制造商将会有更多的创新机会，比如为个人和企业用户带来更加个性化和安全体验的“Windows Hello”。

据介绍，“Windows Hello”让用户能够用更加简便安全的方式通过脸部、虹膜或指纹等生物特征认证来登录设备、网站及服务。同时，这些生物数据将仅在设备上进行本地保存并匿名化，不会与除你之外的任何人共享。“Windows Hello”将通过硬件合作伙伴来提供，他们将推出支持“Windows Hello”的Windows 10设备。

针对时下热门的物联网，微软也做了相应的安排。Windows 10发布后，将首次免费推出小型物联网设备专用的新版Windows。微软宣布Windows 10将推出针对各种物联网设备的版本，从ATM和超声波设备等高性能机器，到网关等资源受限的设备。

Windows 10赚钱

微软推出免费增值策略

一直以来，微软都是依靠售卖软件赚钱，Windows的授权费也是其重要的收入模式，一般一台设备的授权费在5美元-15美元之间，硬件厂商最终将授权费转嫁给消费者。微软财报显示，在消费类市场，微软的营收大头来自设备与消费者授权部门，主要指Windows和Office等软件面向OEM厂商和企业收取的授权费用。

那么在现今的免费时代，微软如何赚钱？微软软件市场营销负责人的答案是——免费增值。据介绍，微软的免费增值策略包括四个部分：获取、吸引、招募和盈利，比如微软推出免费的Office for iPad获取大量的用户，然后吸引用户长期使用该服务，并利用生态的优势将用户圈在服务里，最后再从中找出潜在的付费用户，向他们推广订阅版本。通过免费服务吸引用户，然后推出增值服务，再将部分免费用户转化为收费用户，从而实现收入的增长。

该负责人强调了生态系统的重要性，而不是产品和服务直接相互独立分裂。他以苹果的产品为例，认为苹果将营销费用砸到iPhone等极少的产品中，但这些产品背后却将苹果整个生态系统贯穿在一起，微软也在做相应的探索。

但业内人士分析，由于互联网产品免费的趋势，以及低门槛和廉价的边际成本，免费增值已经成为互联网公司商业模式的主流。但这套模式传统上就不是微软的强项，如何能够将用户圈在微软的服务上是免费增值模式的关键。文/本报记者 吴琳琳

Win10浏览器将启用新名称？

昨天有消息称，微软将在Windows 10系统中内建全新的网络浏览器，这一浏览器开发代号为Spartan，用以取代毁誉参半的IE浏览器，IE品牌也将逐步淡出微软的核心产品线。但截至记者发稿时，微软官方并未给予回应。

消息称，微软行销主管Chris Capossela在Convergence大会上透露，微软公司正在为全新的Windows 系统浏览器设计品牌名称，在 Windows 10 系统中考虑到了老使用者的软件相容性，将会保留 IE 浏览器，但新浏览器将彻底取代IE在Windows 10中举足轻重的地位。在过去几年中微软采用一系列广告和行销活动，扭转IE浏览器在用户心中的糟糕形象，该部门的负责人已经在2014年12月离职，Windows系统浏览器宣告进入新的时代。在全新的浏览器品牌名称中，很可能会加入微软公司的名称。
