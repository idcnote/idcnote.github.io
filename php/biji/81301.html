微软温馨提醒：Win2003即将停止服务

文章正文

Windows XP已于2013年4月彻底结束历史使命，而它对应的服务器版本Windows Server 2003也“行将就木”了。微软今天再次提醒大家，该系统将于2015年7月14日之后不再提供任何支持，包括安全更新、修复补丁、技术支持。Windows Server系列虽然是面向服务器的专业系统，但正因为扎实稳定、高级功能丰富，颇受很多资深自身系统玩家的青睐(就是游戏支持差点)，Windows Server 2003更是凭借轻身材、高效率而一直拥趸甚多，对4+GB内存的支持也是相比于Windows XP的一大亮点，因此诞生整整12年来始终不乏粉丝(生日3.28)。

在它之后，微软还先后发布了Windows Server 2008、Windows Server 2008 R2、Windows Server 2012、Windows Server 2012 R2，以及专门面向家庭应用的Windows Home Server，但产生的影响都不如这位老前辈。

单纯提醒大家一个系统要死了显然太残酷，太没有情调，微软今天就特意为Windows Server 2003写了一篇送别信，称之为好兄弟、好基友，但该分手的时候还是要分手了，还做了几幅很萌的GIF动图，都在哭诉自己已经老了，应付不了如今的局面，是时候休息了。

Windows 10即将诞生，服务器版本也会随之升级，“Windows Server 2015”就要来了。

致 好兄弟：

说好做彼此的基友，手拉手一起走。可是，该分手时就分手。身为一介服务器操作系统，面对更新换代的巨浪，我被拍死在了沙滩上。

2015年07月14日将是我工作的最后一天。到时候，Windows Server 2003技术支持服务将会终止。这意味着，从此之后——你将不能获得关于我的安全更新，我也无法再获得来自微软大佬的技术支持。

我很荣幸可以从03年开始，为你效力至今。这一路，陪你开辟事业理想，陪你度过无数个工作日夜，陪你对付难搞的领导和公司需求，见证了你在IT职场上的蜕变。时光飞逝，你已升职加薪，不再是当年那个黄毛小子。IT领域更是面临爆炸式的革新，就连硬盘里的动作片主角也推陈出新换了好几轮，身在其中，总要与时俱进。十二年了，我也不得不服“老”，自己已经跟不上时代：

Time to say goodbye!

尽管我要离开，但我的爱还在继续！我的接盘侠Windows Server 2012以及Windows Azure将带着最前沿的技术飞奔到你身边！更强大的功能、更好的服务和体验，兄弟！你值得拥有！

在未来，会有更多挑战与机遇扑面而来，愿你与Windows Server 2012或Windows Azure（一起 Zhuang Bi）一起飞，成为更加优（niu）秀（bi）的IT人。

话不多说，我闪了~祝好！
