文章正文

 在 Ubuntu 的各个发行版中，并没有专门发布中文版的 Ubuntu ，如果中国用户想要使用就需要下载中文包以提供系统的中文的支持。而且由于版权等各方面的原因在 Ubuntu 的发行版中也并没有预装功能强大的播放器和解码器。

对于一个对计算机初级用户而言想要完成这几个操作也许存在着一点的难度。不过强化版的 Ubuntu 的出现使这种情况得到了极大的改善。


    今天 Jason  用了好几个小时，终于 remaster 了全新的一张Ubuntu 8.04 安装盘Live-CD。因为加入了很多东西，现在是LiveDVD。

    Ubuntu 8.04 Remastered by Jason

    容量：975.7MB - 1023045632bytes

    MD5 码:  a24b20da3059298e084a03bd3974d775 myUbuntu.iso

    对原版 Ubuntu 8.04 的修改的清单:

    1. 所有软件upgrade到6月17日最新版。所以 Linux 内核也升级为2.6.24-19版。

    2. 加入修正版的 madwifi 驱动for AR5007EG无线wireless Lan. (华硕EEE PC 701~900的wireless card)

    3. 内置中文支持及Scim多种输入法，安装后不用上网download中文support包。但Live-cd模式仍然保留英文环境。

    4. Live-cd模式英文环境中，可使用Fcitx中文输入法。(需要console或alt+f2，输入im-switch -s fcitx，log out再log in)

    5. 已经装好所有视频和音频 codecs，可以播放任何文件。播放器有Totem和SMplayer。

    6. Flash已加入，Java已加入。

    7. 网络软件：eMeSeNe和Skype加入。

    8. 修正Flash, Skype, 与 Totem 的 PulseAudio 冲突问题。

    9. 预装 compiz advanced manager 和 simple ccsm，还有 Emerald。

    10.Firefox 为 3.0 正式版。

    可以在 Windows 下可以用 WuBi 正常安装。

    下载地址1：
     http://www.namipan.com/d/myUbuntu.iso/b151d48459dae66825dfa80495c11596d6cd4dc80070fa3c

    UBuntu 8.04的?u作教程在Live-CD启动后，Home里的Examples的remaster-howto中。

    下载完成后请检查 MD5 校检码

    Ubuntu 8.04 强化版几乎包含了日常所需的应用软件，同时简化了很多繁琐的配置，而且可以通过 WUBI 安装，极大地降低了上手的难度。安装完成后，你几乎不需要进行多少配置便可以满足你的日常使用需要了
