相关函数: $_SERVER['HTTP_ACCEPT_LANGUAGE'];复制代码 代码如下:<?phpheader("content-type:text/html; charset=utf-8");echo '<br />'.$_SERVER['HTTP_ACCEPT_LANGUAGE']
文章正文
相关函数: $_SERVER['HTTP_ACCEPT_LANGUAGE'];
复制代码 代码如下:

<?php
header("content-type:text/html; charset=utf-8");
echo '<br />'.$_SERVER['HTTP_ACCEPT_LANGUAGE'].'<br />';

if(strrpos(strtolower($_SERVER['HTTP_ACCEPT_LANGUAGE']), 'zh-cn') !== false) {
    echo '<br />你的是中文操作系统<br />';
}
else
{
    echo '<br />非中文操作系统<br />';
}
?>
