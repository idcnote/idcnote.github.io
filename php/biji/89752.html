文章正文

微软已经在最新的 Windows 10一月技术预览版中内置Cortana，这也是首个内置Cortana的公开Windows 10版本。在Build 9926中，Cortana只支持美国英语，界面和功能完成度还不高，不过支持基本的笔记本（记录你的兴趣）、语音和文本交互和搜索。

如何开启Cortana？

由于Cortana只支持美国英语，因此需要将系统区域设置为美国，美国英语设置为“主语言”：

• 打开设置>时间和语言>区域和语言>国家或地区里选择美国，语言中添加English (United States)

• 确认设置>系统>语音>口语是English (United States)

• 再去设置>系统> Cortana和搜索>自定义Cortana和搜索设置>新打开的设置面板中将选项切换为开启（On）Cortana

在哪里和Cortana交互？

在上面确认完设置之后，可以在任务栏点击开始按钮右侧紧挨着的Cortana圆圈图标（默认为输入框）开始Cortana。点击后，第一次运行需要进行一系列初始化流程，包括输入自己的名字–即便是你在WP8.1上已经开启过Cortana。

之后再打开类似于WP8.1上的Cortana首页，包括各类兴趣卡片，比如每日通勤、天气、股票、航班、还有相关建议。

Cortana输入框太占地方？

Cortana以搜索框的形式霸占着任务栏较大一块位置，不过可以通过设置缩小占位或者改变按钮位置。第一种方式：

• 任务栏空白处>右键菜单>搜索>选择只显示搜索图标

理论上来说，Cortana按钮的位置是不可拖动，不过可以变通实现挪位：

• 任务栏空白处>右键菜单>搜索>禁用

• 再打开开始菜单>所有应用>找到搜索（Search）>右键，固定到任务栏

之后点击这个任务栏搜索按钮，直接会激活Cortana面板。当然你也可以直接使用开始菜单底部的Cortana搜索框，用法都一样。

如何开启Hey Cortana被动语音激活？

Hey Cortana可以实现不通过点击图标或输入框，通过语音来激活Cortana进行交互。开启方法也不难，直接在Cortana设置：

• 打开Cortana面板>点击左上角菜单图标>设置>最后有选项Hey Cortana >开启

如何获得Cortana个性化内容？

Cortana是通过笔记本（Notebook）来记录用户个性化信息的功能，信息存储在云端，会在手机、平板、PC之间漫游，笔记本里的东西可以是你关注的股票、每日通勤、约会安排、航班、天气等，可以自己去设定或根据Cortana给出的建议去选择。

如果你手机上Windows Phone 8.1版Cortana已经设置过，Windows 10版会自动同步最新的笔记本。你也可以修改：

• 打开Cortana面板>点击左上角菜单图标>笔记本

比WP8.1版Cortana更实用的功能是？

• Windows 10版Cortana支持搜索本地和OneDrive的文件

• 找应用也更加方便，优先在顶部显示（WP8.1版是在网页搜索结果左侧，需要滑动，不容易发现）。

微软Windows 10版Cortana将驱动 Windows 10手机版的Cortana，也将持续更新，微软会不断为其加入新的语言支持、新功能和内容–目前已经在测试的WP8.1版Cortana语言和市场支持也将在之后陆续补上。
