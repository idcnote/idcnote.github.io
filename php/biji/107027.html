文章正文

微软中国昨天向媒体散发了一份博客文章，对国内用户和老外都很关注的盗版用户升级Windows 10问题做出最新说明。根据该说明，对于微软不能确定为正版的用户，当他们通过合作伙伴渠道免费升级到Windows 10后，桌面会出现水印提示，购买正版授权并激活后可以消除提示。

今年早些时候，微软在深圳WinHEC大会上联合360与腾讯对全国Windows用户提供免费升级Windows 10服务，当时我们报道盗版用户一样可以获得升级。这引发了外媒的关注和不安，认为这对正版用户不公平。微软不得不体现出严格打击盗版的姿态以安抚舆论，但直到现在为止，仍未明确堵死盗版用户升级的渠道。

实际上，盗版用户能升级并正常使用Windows 10，微软就可以加速淘汰Windows XP和Windows 7，从而为公司更长远的生态链打造铺平道路。而Windows本身也越来越具有免费化的趋势。

以下为新闻稿（加粗部分为编者所加）：

正版 Windows 和 Windows 10

5.15.2015

这周我们分享了关于 Windows 10 “版本"的更多细节。今天，我想就一些我们收到的关于正版与非正版 Windows的问题给出解答。

随着 Windows 10 的到来，正版 Windows 7 和 Windows 8.1 的客户将可免费升级到 Windows 10（更多详情和条件请见这里）。一旦客户完成升级，他们将在其使用设备的产品支持生命周期内持续获得 Windows 创新功能和安全更新。

这些客户是从我们尊敬的 OEM合作伙伴渠道购买的正版 Windows 授权许可，OEM 合作伙伴可确保 Windows 是合理安装并有相应的授权许可，且未被篡改。Windows 10 为我们的客户提供了更安全、更现代的体验。将我们所有的客户带到 Windows 10 平台上能够帮助开发人员促进这一生态系统的蓬勃发展， 让Windows 的客户体验更出色。我们正在竭尽全力为客户无缝地从 Windows 7 或 Windows 8 升级到 Windows 10而努力。我为Windows 10 升级带来的机遇感到无比兴奋。

当我们无法验证Windows 是否安装并有相应的正版授权许可，且未被篡改时，系统会在计算机桌面上创建一个水印以告知用户。如果你在一台新设备上看到这个水印，我鼓励你立即联系相关零售商并退回该设备，同时要求一台正版 Windows 设备。非正版 Windows会带来更高的风险，包括恶意软件、欺诈及个人信息曝光，以及更高风险的不良使用体验和功能缺失。非正版 Windows 不被微软或信赖的合作伙伴支持。

微软和我们的 OEM 合作伙伴了解许多消费者是在不知情的情况下成为盗版的受害者，随着 Windows 10 的到来，我们希望我们所有的客户与我们共同前进。虽然我们提供的 Windows 10 免费升级将不适用于非正版 Windows 设备，但是我们将会为在非正版状态中运行设备的客户提供 Windows 10。此外，通过与一些 OEM 伙伴的合作，我们计划为运行非正版状态的、较旧设备的客户提供非常有吸引力的 Windows 10 升级计划。请继续关注我们的合作伙伴了解更多从他们提供的升级细节。

Windows 10 仍处于开发阶段，我们尚不能回答所有的问题，但我希望这篇博文能够就一些重要的课题作出相应的澄清。如果你想帮助塑造  Windows 的未来并为 Windows 10 做出贡献，欢迎加入“Windows Insider 计划“，你可以下载 Windows 10 技术预览版，并向我们提出宝贵的反馈。

谢谢你选择正版 Windows 10，来支持 Windows 和 Windows 的合作伙伴。
