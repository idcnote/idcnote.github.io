OS X 10.10 Yosemite的新特性与iOS联系更紧密

文章正文

早在2012年的时候，苹果CEO Tim Cook就曾将微软意欲用Windows统一PC和移动设备的做法比作是“冰箱和烤面包机的融合”，认为把桌面操作系统和移动操作系统合二为一并不明智。虽然目前用户对于Windows 8与Windows 8.1的反馈也证明了这一点，但是桌面设备和移动设备的融合统一还是很有实际意义的，这也正是苹果刚刚正式发布的OS X 10.10 Yosemite的特色所在。

与Windows直接采用针对触控优化的Modern界面不同，Yosemite并没有将注意力放在表面，而是更关注深层次的功能整合，将部分移动设备上的功能无缝整合到Yosemite中，这让原本就非常优秀的OS X变得更加易用。如果你想知道Mac OS X Yosemite究竟有哪些些新特性，不妨一起来看看下面测评体验吧！

全新的外观设计

与之前版本的拟物化设计不同，Yosemite充分借鉴了iOS的扁平化设计风格，看起来更加清新简约，而应用图标配色鲜亮明快，设计趋向抽象化。如果说UI风格是表面功夫，好看与否因人而异，那么系统默认字体的改变则更有实际意义。在Yosemite中，系统默认字体从Lucida Grande变为Helvetica Neue，而后者更适合在高分辨率下阅读。值得一提的是，如果你之前就参加了OS X Beta公测计划并且已经体验过Yosemite，那么一定会非常适应正式版，不过界面设计的改变很快就会让人淡忘。

Continuity让Mac和iOS协同工作

前面我们就提到了Yosemite注重与iOS设备在功能上的整合，而新增的Continuity功能就是最好的例子。仅需一个iCloud账号，Continuity就可以让iPhone用户直接在Mac上接听电话和回复信息，甚至一些其他应用程序的数据也可以进行即时交换，真正实现了Mac和iOS设备的无缝衔接。

毫无疑问，Continuity功能将备受欢迎，想象一下当你操作Macbook时恰巧有电话或者信息，再也不需要腾出手去操作手机，在Macbook上就能轻松处理。不仅如此，用户在Mac正在浏览的网页、正在编写的邮件甚至是iWork文件都可以通过Handoff直接推送至iPhone或者iPad上，让你随时随地都能快速进入工作状态，当然反过来也一样。令人有些遗憾的是，暂时Handoff功能仅支持Safari、邮件、笔记、地图、iWork套件等苹果独有应用，不过如果用户反馈良好的话，今后很多第三方应用也会逐渐加入进来。Continuity的另一项子功能就是即时热点。在没有Wi-Fi时，用户通过Mac即可以自动在iPhone上创建热点，并且能够直接查看iPhone的电池和信号。

不管是Handoff还是即时热点其实都是一些小功能，但正是这些看似不起眼的小功能的存在，才让Continuity变得如此出色使用，也让Yosemite与其他传统桌面系统相比显得那么与众不同。当然，享受Continuity的前提是你至少得拥有一台Mac以及一部iPhone或者iPad。

更好用的信息与Facetime

在Yosemite中，信息和Facetime都经过了全新的设计。现在，信息应用可将普通短信与网络信息相整合，并且还支持群组对话和语音信息，而Handoff可以让这些信息随时都能呈现在用户的任何一台苹果设备上面。

至于Facetime，不仅界面风格与Yosemite相当统一，而且iCloud还会同步用户的通话记录方便用户随时查看。另外，在锁屏界面就可直接接听Facetime邀请，再也不用手忙脚乱地解锁了。让人有些不解的是，全新的Facetime不知为何取消了拨号盘，不过现在在联系人、地图以及Safari中都可直接拨打电话，也算是一种弥补吧！

Spotlight更强大

作为Mac OS的标志性功能之一，Spotlight在Yosemite中自然也得到了升级。当你同时按下Control和空格键时，Spotlight窗口会直接出现在屏幕中央，但在Mission Control界面中不会出现。值得一提的是，Spotlight窗口的位置和大小是固定的，这样做有利有弊。好处是你不需要在很多窗口中去寻找Spotlight，但是坏处是它有时会遮挡用户想看的内容。另外，上次搜索结果会被Spotlight自动保存，方便用户再次调用。

和以前一样Spotlight能够快速调用本地与在线资源，虽然它不是独一无二的，Windows 8.1也具备这项功能，但是Spotlight更优秀的原因是因为它更聪明，很多简单的问题可以直接向Spotlight发问而无需打开特定应用程序，甚至它还能帮你解决简单的数学运算呢！需要注意的是，上面提到的这些只是Spotlight的附属功能，Spotlight的主要作用是帮你准确快速找到Mac或者iCloud上的文件，如果你善用Spotlight，工作效率一定会大大提升。

Safari

Yosemite还带来了全新的Safari浏览器。在界面上，全新的Safari看起来和Yosemite相当协调，缩窄的标签栏使页面内容以更好的方式被呈现出来，而借鉴Spotlight的搜索预览可以显示摘要和缩略图，Safari甚至还能根据iCloud账户中的信息优先推荐那些用户经常浏览的网站。不仅如此，在被重新设计Top Sites页面中，网页预览更长，与大部分网页的垂直滚动布局相吻合，能够呈现更多内容。用户还可以根据需要把Top Sites替换为个人收藏。苹果在发布会上着重提到了全新Safari中类似于Mission Control的抽屉式标签页设计，一个网站下的标签页可以被归为一类，方便用户进行多标签页浏览和管理。

Yosemite将OS X与iOS设备统一协调的设计思路也体现在了Safari中，现在分享按钮不仅可以将网页分享至新浪微博等社交账户，还能通过Airdrop直接与其他苹果设备共享，实用性还是相当不错的。

苹果宣称Safari在性能上是其竞争对手的6倍以上，但是Futuremark Peacekeeper的实际跑分表明，Safari和Chrome在2012年款酷睿i5版Macbook Air上表现基本一致。抛开性能和跑分不说，Safari的实际体验确实要稍微慢一些，虽然在高分辨率下的网页浏览非常顺畅，但是切换和打开标签页有时会出现卡顿，尤其是在多标签页的情况下和Chrome的差距还是显而易见的。

iCloud在云端

Yosemite支持苹果不久前推出的iCloud Drive服务，新用户可以免费获得5GB存储空间，与Dropbox、谷歌、微软等同级别竞争相比还是小了点，要知道这5GB可是包括了iCloud的所有内容，比如如果备份文件占用2GB的话，那么你就剩下3GB来存储文件了。虽然苹果表示iOS设备和Mac都可以访问iCloud Drive，但是别指望其他第三方软件数据会像iWork一样兼容性那么好。值得一提的是，苹果将iCloud在Yosemite的系统层级进行整合，根本无需通过网页就可对iCloud文件进行管理。

在众多新特性中，Family Share家庭分享功能比较容易被人忘记。Family Share可将最多6个iCloud账户绑定在一起，其中的所有家庭成员不仅可以共享在App Store购买的多媒体内容及应用程序，还共同拥有家庭相册、家庭日历以及家庭提醒，而且家庭成员还不仅限于亲人，同事、朋友也可以参与Family Share。

“隐形的”中心

在Yosemite中，通知中心可以从桌面右边推出，但是在实际体验中，用到通知中心的机会其实并不多。首先，大部分通知并不重要，用户往往会选择忽略，即使需要处理也无需打开通知中心。其次，诸如短信、电话以及社交消息这类相对重要的通知，用户一般都会在iPhone或者iPad上及时处理，毕竟Yosemite是桌面操作系统，而通知中心对于移动操作系统来说显然更重要。不仅如此，天气、股票、日历等小插件也并不实用，大部分情况下用户更偏向于使用完整的应用程序，而唯一有用的计算器通过Spotlight来呼出其实更简单。虽然Yosemite的通知中心显得有些多余，但是如果开发者能推出第三方插件的话，相信通知中心的尴尬现状会有所改变。

总结

总体来说，OS X 10.10 Yosemite是一款非常出色的桌面操作系统但它并不完美，虽然Continuity、全新Spotlight以及Safari带来的系统原生体验要远超Windows，但是略显鸡肋的通知中心以及紧张的iCould Drive免费空间还是让人稍感遗憾。尽管如此，Yosemite显然还是要优于进步幅度并不大的Windows 8.1，绝对值得升级，更何况它还是免费的呢！

优点：

n 新特性实用

n 增强与iOS设备连接性

n 免费

n 优秀的桌面操作系统体验

缺点：

n 通知略显中心鸡肋

n Safari多标签页体验欠佳

n 5GB默认iCloud Drive空间偏小
