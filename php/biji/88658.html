文章正文

升级win10正式版后，发现电脑经常会失去响应，且提示“a language pack isn\'t avaliable”，这该怎么办呢？下面phpstudy的小编就为大家介绍下具体操作步骤。

原因分析：

其实，该问题可能是因为你的系统的默认语言是英文导致的，改成中文就好了。

解决方法:

1、如果你还没有在输入法里添加简体中文，请先在开始，settings（或者按win+I），time & language，region & language， add a language，选择中文简体，选择中国，add。

2、然后在控制面板（win+X，选择control panel），选择language或者add a language，在中文输入法后面选择option，你就会发现这个错误。

3、Press WIN+R to bring up the Run box – type Regedit。

4、Then in the Registry, find the pathHKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate

f)       There should be one or two keys there

a.       WUServer

b.      WUStatusServer

>>> delete these 2 entries.

5、点击开始，输入Command Prompt，在搜索出来的程序右键，选择Run As Administrator。

6、输入Net stop wuauserv，按enter。

再次输入Net start wuauserv，按enter这个步骤是重启windows update。

7、在settings里选择update & security，点击右边的check update或者retry。

8、再进行第二步，发现变成了check availabiliy。（如果没有变成这个，还是原来的错误a language pack isn't avaliable，请去玩20分钟游戏，再次进行第二步操作，查看此处是否变成check availabiliy）

9、重启机器，再进行第二部，会看到原来的a language pack isn't avaliable，变成了install相应的语言。点击安装，一步步操作即可。

10、再次回到第二部，发现多了一个选项，make this the primary language，选择这个选项，log off now。

11、再次登陆（登陆时间可能较长），好的，系统就变中文系统啦。

以上就是对win10系统升级后电脑经常无响应提示a language pack isnt avaliable的原因及解决方法全部内容的介绍，遇到win10系统提示“a language pack isn\'t avaliable”问题的朋友们，只要按照上述步骤操作一遍就可以了。
