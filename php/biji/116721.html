循环用于重复执
文章正文

这篇文章主要为大家详细介绍了Python循环的简单示例，具有一定的参考价值，可以用来参考一下。

对python这个高级语言对此感兴趣的朋友，看看idc笔记做的技术笔记！

循环用于重复执行一些程序块。从上一讲的选择结构，我们已经看到了如何用缩进来表示程序块的隶属关系。循环也会用到类似的写法。

for循环

for循环需要预先设定好循环的次数(n)，然后执行隶属于for的语句n次。

基本构造是

for 元素 in 序列:

statement

举例来说，我们编辑一个叫forDemo.py的文件


# @param Python基础教程06 - 循环
# @author php教程|www.512Pic.com 

for a in [3,4.4,'life']:
    print a

# End www_512pic_com

这个循环就是每次从表[3,4.4,'life'] 中取出一个元素（回忆：表是一种序列），然后将这个元素赋值给a，之后执行隶属于for的操作(print)。

介绍一个新的Python函数range()，来帮助你建立表。


# @param Python基础教程06 - 循环
# @author php教程|www.512Pic.com 

idx = range(5)
print idx

# End www_512pic_com

可以看到idx是[0,1,2,3,4]

这个函数的功能是新建一个表。这个表的元素都是整数，从0开始，下一个元素比前一个大1， 直到函数中所写的上限 （不包括该上限本身）

(关于range()，还有丰富用法，有兴趣可以查阅， Python 3中， range()用法有变化，见评论区)

举例


# @param Python基础教程06 - 循环
# @author php教程|www.512Pic.com 

for a in range(10):
    print a**2

# End www_512pic_com

while循环

while的用法是

while 条件:

statement

while会不停地循环执行隶属于它的语句，直到条件为假(False)

举例


# @param Python基础教程06 - 循环
# @author php教程|www.512Pic.com 

while i < 10:
    print i
    i = i + 1

# End www_512pic_com

中断循环

continue # 在循环的某一次执行中，如果遇到continue, 那么跳过这一次执行，进行下一次的操作

break # 停止执行整个循环


# @param Python基础教程06 - 循环
# @author php教程|www.512Pic.com 

for i in range(10):
    if i == 2: 
        continue
    print i

# End www_512pic_com

当循环执行到i = 2的时候，if条件成立，触发continue, 跳过本次执行(不执行print)，继续进行下一次执行(i = 3)。


# @param Python基础教程06 - 循环
# @author php教程|www.512Pic.com 

for i in range(10):
    if i == 2:        
        break
    print i

# End www_512pic_com

当循环执行到i = 2的时候，if条件成立，触发break, 整个循环停止。

总结

range()

for 元素 in 序列:

while 条件:

continue

break
