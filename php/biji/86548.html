如何在苹果电脑上使用淘宝旺旺购买商品

内容摘要
虽然在淘宝上购买商品大都是屌丝们的事情，但是如果使用苹果电脑的高端友友也想使用旺旺，是不是有些不方便呢，那就先学习下淘宝官方的旺旺使用教程吧。
浏览器上面点击旺旺聊天
文章正文

虽然在淘宝上购买商品大都是屌丝们的事情，但是如果使用苹果电脑的高端友友也想使用旺旺，是不是有些不方便呢，那就先学习下淘宝官方的旺旺使用教程吧。

浏览器上面点击旺旺聊天的图标 () 后无法打开旺旺客户端？

如果你在网页上面点击旺旺聊天功能没有响应，你可以尝试手动下载 旺旺浏览器插件 并安装，然后重新启动浏览器就可以了。 (Safari 或 Chrome 都支持, 注意：插件在 Safari 浏览器下面可能会被 ClickToFlash、ClickToPlugin 扩展阻止加载，你需要在“扩展设置”内打上勾过滤掉)

如何记住密码?

点击密码输入框前面的钥匙图标，变为蓝色时登录后可自动记住密码。

切换登录过的帐号?

如果你登录过多个帐号，在登录界面中点击头像，可以选择以前登录过的帐号，也可以删除不想保留在电脑中的帐号。

如何添加联系人或群?

直接通过主界面或系统栏里的搜索框，输入旺旺 ID 或群号，点击在网络中搜索，可对搜索出的人或群做相应操作。

如何切换回传统的聊天形式?

打开阿里旺旺偏好设置（程序菜单栏：阿里旺旺 -> 偏好设置），在会话 tab 中选择传统模式。

程序的快捷键有哪些?

快速查找联系人 ⌘⇧F

Tab 切换 下一个 ⌘⇧] ,前一个 ⌘⇧[

关闭当前Tab ⌘W

联系人列表如何切换到小头像显示?

右键点击任意联系人，然后在弹出菜单中选择切换到小头像显示

如何删除历史消息?

在消息管理器中，选中一个联系人或者群组，然后点击工具栏上的删除，则会提示你是否要删除消息。 如果你想删除和某人聊天的某一条，可以将鼠标移到某条消息上，然后点击 X 可删除当前消息。

如何屏蔽旺旺群消息或者系统消息?

右击某个群组或系统消息，在弹出的菜单中选择群消息设置->不接收

如何查看卖家店铺信息?

点击聊天窗口右下角店铺图标，会弹出界面显示卖家信用，店铺链接等信息。
