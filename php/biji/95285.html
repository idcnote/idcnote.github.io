苹果OS X Yosemite 中怎么使用Markup功能？

文章正文

Markup 是 OS X Yosemite 中一个利用不足的功能，甚至有很多人都不知道它的存在。你可以在 OS X 的邮件程序中用 Markup 来对你附件中的图片或 PDF 邮件进行简单的编辑与注释。这个工具的使用非常简单，只要你学会了，就可以节约很多时间并且能让你更加富有成效。下面小编就来教你怎么使用它。

1、激活 Markup

Markup 功能应该是默认为打开的，但你开始使用前最好确认一下。

第一步，打开苹果菜单，选择系统偏好设置。

第二步，选择扩展面板，点击“操作”，然后你就可以看到一个名叫“Markup” 的复选框，在继续之前确保选项已经勾选了。

2、隐藏在邮件中的Markup

        正如上文提到的那样，这个工具是在邮件程序中使用的，但你不会在邮件里找到任何关于它的菜单选项。要使用这个工具，你需要写一封新邮件或回复一封邮件，并且这封邮件里一定要有图片文件或 PDF 文件。

        激活 Markup 的步骤如下：把光标放在附件上，右上角会显示一个小的图标，点击该图标，打开一个菜单，选择 “Markup”选项，这时邮件就会打开一个新窗口，新窗口中会显示邮件信息和附件，顶上会出现 Markup 工具条。

3、Markup 工具

工具条一旦激活，你就能在上面看到以下选项：

    •    素描: 素描工具能让你在图片或 PDF 上涂鸦。如果你画的图形接近某种标准图案，比如说一个圆或一个三角形，系统就会把它自动修正为标准图案。

    •    图案: 这个工具能让你从已有的图形中选择一个，放在文件的任何地方。

    •    文本: 可以在文件任何地方加入文本框，用户可以在文本框中作备注。

    •    签名: 让你可以自动添加签名。

    •    图案风格:  这个菜单可以让你编辑图案的风格，选项包括线条的宽度、阴影的范围、箭头的方向。

    •    边框颜色: 让你快速便捷地调整图案的边框颜色。

    •    色彩填充: 你可以自定义图形内部的颜色。

    •    文本风格: 该选项能让你调整字体风格、大小、颜色、对齐方式。

虽然每个 Markup 工具的使用都非常简单，但有时候会很繁琐，这需要练习，不过只要你掌握了诀窍，它就会变的非常有用。

4、获取图片

      Markup 还有一个非常好用的工具：你可以把鼠标移到某个图像的边缘，然后会出现一个抓取图标，你可以用它快速获取图片，而不用把图片下载到电脑上再用另一个软件来查看。这个工具用起来相当顺手。
