成功之后mysql根目录下才会出现data文件夹，原本应该是没有的！

绕过密码验证：

代码如下：

再另外使用管理员权限开一个cmd窗口，

切换到mysql的bin目录下启动mysql，

代码如下：

然后使用命令进入管理界面

进入后再使用以下命令修改root密码，请注意！Mysql8.0修改密码和之前的不一样！

代码如下：


ALTER user 'root'@'localhost' IDENTIFIED BY '新密码';

之后刷新一下就可以重启正常使用了：

代码如下：


flush privileges;

MySQL8.023安装过程图文分析（首次安装）

以上步骤所使用到的命令应该有以下几个：

代码如下：


mysql -install //安装mysql
mysql --initialize-insecure --user=mysql //初始化数据文件
net start mysql //启动mysql服务
mysql -u root -p //使用无密码进入修改root密码
ALTER user 'root'@'localhost' IDENTIFIED BY '新密码'; //修改root密码，Mysql8.0方式
flush privileges; //刷新数据库
exit //退出mysql管理界面
net stop mysql //关闭mysql服务
