6. 增加一个列

8.自增id属性从1开始每次加1

9.创建索引

create index 索引名 on 表名(列名)

10.删除索引

12.case when的用法（修改表名称id，当 t 的名字不为空,则还是 t 的名字,否则变为表名称的名字，把被修改的数据输出到临时表）

代码如下：


update pn
set pn.id=case when t.id>'' then t.id else pn.id end
output deleted.id into 临时表
from 表名称 pn with(nolock)
join #temp t
