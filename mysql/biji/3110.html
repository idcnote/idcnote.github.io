解决办法：

分析下来是由于某些特殊的改动导致了SQL SERVER2005数据库网络协议中VIA协议的启用，讲该协议关闭即可。Sql Server Configuration Manager-------Sql Server 2005网络配置------MSSQLSERVER的协议，讲VIA协议禁用即可。【图片暂缺】
