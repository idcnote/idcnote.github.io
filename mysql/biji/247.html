感兴趣的小伙伴，下面一起跟随数据库教程的小编来看看吧！

1.先检查系统是否装有mysql

代码如下：


rpm -qa | grep mysql

CentOS7.5 安装MySQL的教程

2.下载mysql的repo源（5.7）

代码如下：


wget -i -c http://dev.mysql.com/get/mysql57-community-release-el7-10.noarch.rpm

CentOS7.5 安装MySQL的教程

如果报错:-bash: wget: 未找到命令

安装插件yum -y install wget

3.安装 mysql-community-release-el7-5.noarch.rpm包

代码如下：



sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm

CentOS7.5 安装MySQL的教程

4.安装MySQL

代码如下：



sudo yum install mysql-server

CentOS7.5 安装MySQL的教程

5.重置MySQL密码

查看MySQL运行状态，运行状态如图：

代码如下：



systemctl status mysqld.service

CentOS7.5 安装MySQL的教程

此时MySQL已经开始正常运行，不过要想进入MySQL还得先找出此时root用户的密码，通过如下命令可以在日志文件中找出密码：

代码如下：



grep "password" /var/log/mysqld.log

CentOS7.5 安装MySQL的教程

如下命令进入数据库：

代码如下：



mysql -uroot -p

CentOS7.5 安装MySQL的教程

输入初始密码（是上面图片最后面的 Es9akjRtrd<4），此时不能做任何事情，因为MySQL默认必须修改密码之后才能操作数据库：

代码如下：



ALTER USER 'root'@'localhost' IDENTIFIED BY 'new password';

CentOS7.5 安装MySQL的教程

其中‘new password'替换成你要设置的密码，注意:密码设置必须要大小写字母数字和特殊符号（,/';:等）,不然不能配置成功

6.重启MySQL服务

service mysqld restart

接着设置Root账户远程连接密码，账户和密码都是 root

代码如下：



 mysql -u root -p
