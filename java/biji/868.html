文章正文

break、continue、return的区别：

break：默认是跳出最里层的循环，也就是break所在的最近的那层循环。

continue：是终止本次循环，继续下次循环。

return：结束当前方法。

3层循环的简单测试：

for (int i = 0; i < 3; i++) {
    for1:
    for (int j = 0; j <3; j++) {
        for2:
            for (int m = 0; m < 3; m++) {
                for3:
                    System.out.println(i+"--"+j+"--"+m);
            }
        }
    }

简单测试break：

    for1:
    for (int j = 0; j <3; j++) {
        for2:
            for (int m = 0; m < 3; m++) {
                for3:
                    if (m == 1) {
                        break;
                    }
                    System.out.println(i+"--"+j+"--"+m);
