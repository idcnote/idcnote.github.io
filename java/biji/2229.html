1、switch语句

int a = 1,b =2;
switch(a+b){
	case 1:
	System.out.print(1);
	case 3:
	System.out.print(3);
	case 4:
    System.out.print(4);
    default:
    System.out.print(5);
}

1、先执行 a+b 得出值 3

2、找到相对应case 3，然后继续向下

3、执行执行所有的语句，因为没有 break

2、循环语句

求1000以内的素数

int j;
for (int i = 0; i < 1000; i++) {
	for (j = 2; j < i; j++) 
		if (i % j == 0)
			break;
    if (j == i)
    	System.out.println(i);
}

结果：

2
3
5
…

当然上面犯了一个明显的错误，最外层的循环应该是<=1000,虽然并不影响什么，但要铭记。

for (int i = 0; i < 1000; i++) {
	if(i == 2)
		System.out.println(2);
    for (int j = 2; j < i; j++) {
    	if(i % j == 0)
        	break;
        if(j == i - 1 )
            System.out.println(i);
     }
}
