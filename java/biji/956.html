java技术

java技术是SUN公司开发的一种编程语言，同时也是一个平台。其主要优点有：面向对象，可移植性，安全，多线程，高性能，跨平台，更重要的是开源，这就成就了java的今天。无数的java爱好者活跃在世界的每个角落。当然，java技术也有它的缺点，它很多帮助文档是英文，这就对学习者的英文水平有一定的要求，另外，java技术不能真正实现跨平台，有很多局限性。 【推荐教程：Java教程】

据有关统计数据显示，java技术人才目前在我国的市场缺口高达500万余人。目前社会上能够符合要求的java程序员还是匮乏，行业处于供不应求的情况。据在所有软件开辟类人才的需要中，对Java工程师的需要到达全数需要量的60%~70%。可以说比起其他专业的求职者来说，Java技术人员的就业前景要广阔的多，而且薪资待遇也比较丰厚，尤其是有一定经验的java程序员，其待遇要更加的好，而且非常吃香，各种企业抢着跑来橄榄枝。所以java技术的前景还是非常好的。

.net技术

.net是微软公司的一个编程语言，其主要优点有：跨语言跨平台，安全，以及对开放互联网标准和协议的支持。.net支持多种语言的互操作，即在一种语言下开发的组件可在另一组件下，通过面向对象的继承而得以重用。目前.net支持的语言达二十多种。而微软公司所做的有中文版本的帮助文档，不用担心英语水平。.net入门比较容易，学习周期短。.net的缺点就是不能免费，现在市场的需求量没有java大，这也形成了.net的短缺。

由于越来越多的企业纷纷采用.net技术开发，或将现有产品移植到.net平台上，使.net的应用空前广泛。.net的快捷开发，高效率，低成本的特性，已经真正给从事.net的开发人员带来了令人欣喜的机会和强大的竞争力。市场对.net开发人员的需求越来越大，在各大职业网站上，.net的开发人员的职位数量已远远超出其他职位。

Java和.net哪个好？这个其实是没有一个好坏之分的。学习计算机编程语言，无论是java，还是.net，主要的还是自身的水平，任何一个语言都有良好的前景，前提是自己掌握真正的技术，没有哪个是不好的。

如果想追求高薪高就业广的话学java还是相对来说靠谱点，在全国很多城市需求都很大，包括一些二三线城市。
