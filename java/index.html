Java教程

  1. 首页 > Java教程
Java 是一门面向对象编程语言，不仅吸收了 C++ 语言的各种优点，还摒弃了 C++ 里难以理解的多继承、指针等概念。

这套Java教程针对有编程基础的读者,既有入门知识,也有进阶知识,Java学习人员借助这套教程能够很多掌握Java的常用技能。

本教程通过简单的实例将让大家更好的了解 Java 编程语言。 移动操作系统 Android 大部分的代码采用 Java 编程语言编程。

学习Java基础不但可以用来开发网站后台、桌面应用程序、Web应用程序、分布式系统、嵌入式系统应用程序和 Android APP，还在数据分析、网络爬虫、云计算领域大显身手。
