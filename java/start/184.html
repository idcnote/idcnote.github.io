文章正文

下面我们来认个眼熟，看看最简单的java程序长什么样子

/*
*第一个java程序
*/
public class Main{
    public static void main(String[] args) {
        System.out.println("Hi www.dotcpp.com");
    }
}

大家可以在dotcpp的在线编译器上运行这段代码，可以看到会输出字符串“Hi ，这就是这一小段代码运行的效果，下面来给大家详细介绍：

首先我们看到输出这个字符串的语句是一个System.out.println的语句，其实我们可以修改这个语句中双引号里的内容，修改什么就会输出什么，也就是说这个是个输出语句，我们只要把想输出的信息放到这个语句后面的双引号里就可以输出，大家可以多多尝试。

其次，我们看到printfln这个语句又在一个public static void main的小括号括着，这个是个叫main的方法，它是程序的入口点，也就是说全部的代码从这里开始执行，pirntfln则是这个方法中的唯一一条语句，而main方法前面的public static表示这个main方法是一个公开访问的静态方法。

 

然后接着再往外层看，public static void main这个函数又在public class Main这个小括号的里面，public class Main则是一个类，同样public表示这个类的访问权限是公开的，class是一个关键字表示类，后面Main是类名

大家可以尝试理解这一小段代码，并不需要有理解的压力，因为后面还会大量的使用和深入理解慢慢加深。
