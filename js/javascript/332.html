javascript跟随滚动效果插件代码（javascript Follow Plugin），
有需要的朋友可以参考一下Js 跟随滚动效果插件支持定义多个跟随ID，
采用css fixed属性，不支持ie6，兼容其他主流浏览器。支持定义滚动到底部的最小高度，
不会覆盖底部页面大小resize后，插件会自动重置参数=======Js 跟随滚动效果插件
1.支持定义多个跟随ID，采用css fixed属性，不支持ie6，兼容其他主流浏览器。
2.支持定义滚动到底部的最小高度，不会覆盖底部
3.页面大小resize后，插件会自动重置参数=======使用方法 
<script type="text/javascript" src="follow.js"></script>
/**
 * 跟随滚动效果
 *
 * @param 
 * @arrange （www.idcnote.com）
 **/
window.onload = function(){
var followIds = [document.getElementById("follow"),document.getElementById("follow2")];
new Follow({
obj:followIds,
bottom:150
});
}
