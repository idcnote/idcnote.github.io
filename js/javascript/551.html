今天看到两个iframe页面同时滚动的方式，觉得很方便，就找了代码，在这里整理一下：

主页面 index.htm 代码：


< body>
  < table border="1" cellpadding="0" cellspacing="0" width="300" height="400">
    < tr>
      < td height="400" width="50">< iframe name="left" src="left.htm" width="50" height="100%">< /iframe>< /td>
      < td height="400">< iframe name="right" src="right.htm" width="100%" height="100%">< /iframe>< /td>
    < /tr>
  < /table>
< /body>
