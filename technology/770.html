Windows 8对比Windows RT 两者区别解析

内容摘要
本月晚些时候即将发布的Windows 8操作系统将带给我们自Windows 95系统发布以来Windows系统最大的改变。而这款最新的Windows操作系统以三个关键版本的形式发布。
文章正文

本月晚些时候即将发布的Windows 8操作系统将带给我们自Windows 95系统发布以来Windows系统最大的改变。而这款最新的Windows操作系统以三个关键版本的形式发布。

Windows 8是标准版，而Windows 8专业版则为专业用户提供了更多的功能。Windows 8 RT（RT是Runtime的简写）则完全不同，这是一款全新的操作系统。你会为你的PC和Windows 8平板电脑安装哪一款系统呢？Techradar网站日前就Windows 8和Windows 8 RT两款系统进行了对比，看完对比你也许会有答案。发布日期

Windows 8预计的发布日期在今年的十月，而Windows RT和Windows 8平板电脑则会在今年11月发布。

购买方式

Windows RT操作系统只能通过预装到电脑上来获得，你无法单独购买。而Windows 8会预装到新的电脑上，同时也会为消费者提供零售的盒装版和升级版。

价格

从字面角度去看，Windows RT是无价的，因为你不能通过零售的渠道单独购买这款操作系统。但是制造商为其支付的许可费应该显著低于Windows 8和Windows 8专业版。预装有Windows RT的设备预期销售价格将会比预装有Windows 8操作系统的设备要便宜（这样定价的原因考虑到了Android平板电脑和iPad对于Windows销售的冲击）。除此之外，Windows RT的用户将免费获得Office办公软件系统。

升级

你不能从任何一款操作系统升级到Windows RT，因为它是一款全新的操作系统，而且不支持微软之前发布的操作平台。

如果你现在使用的是Windows 7操作系统，那么你可以从Starter、Home Basic和Home Premium这三个版本直接升级到Windows 8标准版。如果你安装的是Windows 7旗舰版或者专业版，那么在升级之前，你需要先执行一次卸载以保证系统的安装是干净的。

如果你想要升级到Windows 8专业版，你也可以通过任何一个版本的Windows 7操作系统进行升级。同样的，32位的系统安装版本只能升级到32位的Windows 8专业版，64位的系统安装版本只能升级到64位的Windows 8专业版。

硬件支持情况

硬件支持情况是Windows RT和Windows 8这两款系统最大的区别：Windows RT只能在配置有ARM处理器的设备上运行，而Windows 8只能在x86设备上运行。Windows RT不能在装有英特尔或者AMD处理器的电脑上运行，而Windows 8也不能在配置有ARM处理器的设备上运行。这就意味着，Windows 8将不支持新的Surface平板电脑。

界面

Windows RT和Windows 8这两款系统都采用了新的Metro界面。运行一些旧的应用程序时，Windows 8系统能切换到传统的桌面（Desktop）模式，Windows RT做不到这一点。但是Windows RT也为一些微软的应用软件预留了Windows Desktop模式，但是对于第三方软件来说，它们将不能使用这种模式。

软件和兼容性

软件和兼容性是Windows RT和Windows 8这两款系统另一个重大的区别。Windows RT操作系统默认自带了Office系列办公软件，而Windows 8标准版和专业版则没有这样的待遇。但是，Windows 8标准版和专业版可以同现有的Windows程序实现兼容，但是Windows RT不具备这样的兼容性。Windows RT操作系统上的应用软件必须使用metro界面进行操作，而Windows 8除了metro界面之外，还可以切换到Windows desktop模式。

Windows RT还对APIs（应用程序编程接口）开发者的使用权限进行了约束，特别是在网络浏览器方面。比如，看起来Firefox浏览器不能推出Windows RT可以使用的版本，因为Mozilla的法律总顾问Harvey Anderson称：“只有Internet Explorer浏览器可以运行许多对于现代浏览器的速度、稳定度和安全性至关重要的先进的计算功能，而用户也习惯于使用Internet Explorer浏览器。”

功能

正如我们预期的那样，不是所有Windows 8具备的功能在Windows RT都能得到体现。Windows RT操作系统中没有Windows Media Player、Storage Spaces、Bitlocker加密技术（虽然Windows RT提供了设备加密作为替代）、Group Policy管理器和域名支持功能。而且Remote Desktop只有在客户模式下才可以工作。

Windows RT和Windows 8都配置了多语言支持、IE10、Xbox Live、Windows Defender、Exchange ActiveSync、Windows的Play To流媒体和虚拟私人网络。
