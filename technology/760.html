Windows 8系统和Windows RT有什么区别？

内容摘要
Windows 8和Windows 8新设备将于本月26日与全球消费者见面，不过，用户在购买这些设备或软件升级包之前，应充分了解Windows 8不同版本间的区别。
文章正文

Windows 8和Windows 8新设备将于本月26日与全球消费者见面，不过，用户在购买这些设备或软件升级包之前，应充分了解Windows 8不同版本间的区别。数月前，微软宣布将为其即将上市的操作系统Windows 8推出3个消费级版本：Windows 8、Windows 8 Pro和Windows RT。
Windows 8和Windows 8 Pro之间区别不大，它们都拥有全功能开始屏幕和桌面环境，而这两者与Windows RT之间的最大区别在于Windows RT运行的是移动环境，且仅运行Windows Store上的应用程序和预装软件。

Windows 8和Windows RT看上去几乎相同，采用新名称的开始屏幕用户界面和可用的桌面模式。不过，深层次的区别对用户如何使用设备有着重大的影响，Windows RT是专门针对平板电脑和移动设备的操作系统，而Windows 8则是全功能PC操作系统。尽管它们看上去好像没有什么区别，但是运行起来却完全不一样。微软一直在努力让其商店零售人员了解Windows 8和Windows RT之间的区别。不过，这可能需要时间。即便是现在Surface RT已经开始接受预订，许多微软员工也无法解释Windows RT到底能做什么和不能做什么。

以下是对Windows 8和Windows RT进行的对比：

发布日期

Windows 8和Windows RT的发布日期均为10月26日。

价格

Windows 8和Windows 8 Pro将可以在零售店和网上购买，单独升级。针对Windows 8 Pro的物理升级包已经接受预订，售价为70美元。微软表示，用户在10月26日还可以在线升级至Windows 8 Pro，费用为40美元。在2012年6月2日至2013年1月3日购买Windows 7设备的用户将可以支付15美元升级至Windows 8 Pro。不过，微软并没有公布基础版Windows 8的价格。

Windows RT从技术上讲属于非卖品，因为用户无法购买到Windows RT升级包。Windows RT将仅是预装在某些特定的设备中，如Surface RT和微软原始设备制造商合作伙伴的产品。预计Windows RT设备的价格会低于Windows 8设备。

硬件

Windows 8将仅在由英特尔或AMD芯片支持的x86架构设备上运行。

Windows RT将仅在ARM架构设备上运行。ARM芯片功耗更低，价格更便宜。

电池续航能力

即将推出的Windows 8设备的电池续航能力估计在6到8个小时。

Windows RT设备的电池续航能力优于多数Windows 8设备。微软表示，Surface RT的电池可以用上一整天，续航能力在8到13个小时。戴尔也声称其即将推出的XPS 10 RT平板电脑的电池续航能力可达20个小时。

界面

Windows 8使用的是全功能开始屏幕和桌面模式。

乍一看，Windows RT的界面与Windows 8的相同，不过，Windows RT的桌面模式并非是全功能的。事实上，用户无法在Windows RT的桌面上运行第三方软件。

软件支持

Windows 8将能够运行Windows Store中的所有软件和以前Windows版本运行的所有第三方软件。

Windows RT将仅支持Windows Store中的应用程序和预装版本的Office 2013。这就是说用户将无法下载任何来自网络在桌面环境下运行的软件。Windows RT不支持第三方浏览器，不支持第三方游戏，不支持第三方的任何软件。用户在Windows RT上运行的软件必须经微软认证且来自Windows Store。

目前，Windows Store拥有约5000个应用，超过90%的应用可与Windows RT兼容。这个应用数量并不庞大，且像Spotify和Facebook等一些重要应用也无法在Windows RT上运行。Windows RT不具备Windows Media Player功能，但预装了如Xbox Music等其他媒体播放器。

企业应用

Windows 8是从Windows以前版本演化而来，支持企业用户必需的工具。Windows 8 Pro提供远程桌面服务器、活动目录域支持、文件系统加密、虚拟化和BitLocker等功能。

Windows RT不一定是企业友好平台，因为它无法兼容第三方软件，用户将必须依靠预装的微软应用来完成工作。预装的RT版Office 2013将没有像Outlook这样的企业常用软件，但提供了Word、Powerpoint和Excel软件。像Surface RT这样的Windows RT设备可能会成为另一个非常好的企业工具，但是如何用户希望拥有全功能的企业平台，那么则必须选用Windows 8。

用户不同

如果用户想要带有真正有用桌面模式的全功能操作系统，那么就应该选择Windows 8。如果用户依赖于第三方软件，那么应该选用Windows 8或Windows 8 Pro。笔记本电脑、台式电脑等设备都可以使用该系统。

如果用户想要一个具备如iPad功能的平板电脑或移动可转换设备，那么选用Windows RT设备非常合适。以Surface RT为例，该产品足可以满足那些想拥有Office等基础生产力工具，以及触摸和移动友好环境的用户需求。不过，请记住，Windows RT仅能运行Windows Store中的应用，目前Windows Store中的应用并不十分丰富。
