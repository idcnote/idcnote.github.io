在Win XP中处理数码照片(上)

内容摘要

Windows XP有个称为："digital camera friendly"的向导，中文含义是"数码相机之友"．作用是一步步引导用户处理照片。包括数码照片或扫描仪扫出的图像
文章正文

    Windows XP有个称为："digital camera friendly"的向导，中文含义是"数码相机之友"．作用是一步步引导用户处理照片。包括数码照片或扫描仪扫出的图像文件．

　　如果这种处理过程相当于旅游一趟的话，这个旅途上好看的"景点"有：

　　1. 数码照片文件预处理．
　　2. 使用数码照片显示探测器．
　　3. 数码照片打印．
　　4. 将数码照片发布到WEB页面（所谓WEB出版向导）．

　　本操作所用的设备：
　　第一：Sony Vaio R505TSK笔记本电脑．配置：(850Mhz PIII, 256MB RAM, 20 GB HD)．

　　第二：数码相机．型号任意，但要求接口合适．

　　临行之前，还得交待几点，作为“旅游须知”：

　　（1）windows xp的图标已经与原先版本的windows不一样了。主要有两大改进：一是图标尺寸更大，支持的显示颜色模式更高，并进行了很细致的美化；二是由于有了前面的基础。使得用户可以自己定制图标。桌面上表示程序、文件夹等项目的图标，原来只能用系统提供的，即使可以更改，也被限制在系统允许的范围之内。到windows xp，允许用户自己定制图标，以充分体现个性化特征．最使人称道的就是，可以将照片当做图标用。例如，输入你自己的照片，然后用这幅照片代表专属于你的使用环境或你自己专用的文件夹．这种功能，在过去的windows产品中是没有的。不过这也带来了负作用：图标和照片的尺寸及显示效果都一样，用户将难以分辨出图标和照片。为了易于区分，规定凡是图标，某个角上加入一个打勾的小方框．这样，即使屏幕上出现了两幅相同照片，立刻就会分辨出哪个是作图标用的．下面看的时候，不要认错了（认错人的"体验"可是很尴尬的呀！）．

　　（2）"digital camera friendly"所能处理的，并不仅局限于数码照片．别的图片也可以处理．只要是它能兼容的文件格式，是可以"通吃"的．

    第一景点：数码照片文件预处理

　　这一站，我们可看到的景致是，如何将数码相机中拍的照片，"灌"到电脑中，并进行正式打印前的预处理：

　　首先要将储存图像文件的设备与计算机进行物理连接．两者究竟如何连接，要看所用的储存设备类型，且主要是看储存设备的输出接口的类型．常见的设备都与电脑有标准的接口，例如USB接口（包括使用USB接口的移动硬盘）．若有USB接口，用起来就简单多了．如果图像文件保存在储存卡上，接口又不通用，则必须经过接口变换．总之，物理连接完成后，才可以启动计算机．一般说，系统会自己找到刚才新增添的设备，windows xp的向导也会热请地自行开始投入为你服务的工作．下图是我们看到的第一幅风景：打开储存图像文件的磁盘：

 

　　这是个对话框，让你选择下一步操作：共有4项选择：

　　1."释放"照片：将照片的原始文件从数字相机中“取”出来；
　　2.以幻灯片形式显示照片图标――或者说将照片以图标的形式显示；
　　3.打印照片；
　　4.打开原有照片的文件夹．

　　逐项解释如下：

　　在数码相机中已经储存有拍好的照片的条件下，可以选择释放照片操作。

　　鼠标点击选择第一项作业："释放"照片．

　　首先指定要释放的照片位于何处？这里其实就是指定所连接的数码相机――已经与主机连接成功的数码相机在屏幕上被显示为一个文件夹。双击该文件夹，就可以打开它。然后将会出现下面的对话框：

 

　　这个对话框中是包含要选择处理的照片以及项目：

    其中的的照片都被显示为缩小的模式。

　　凡是该文件夹中可以处理的照片，都显示在对话框的中间，可以用鼠标单击来进行选定，选一幅或是选多幅处理．依照你的需要定。中间区域下面有三个按钮，前两个指定对照片的旋转操作，后一个用来显示照片的属性信息．一般说，有两种情况需要使用旋转操作：一是想得到特殊效果．二是在打印前的版面设计．至于照片文件的属性显示作何用途，我们后面会给出解释．不论哪种需要，都只需点击相应的按钮就可以执行．

　　现在该进行将照片文件从外面“释放”到主机中这一步了！

　　也就是我们看到的下一幅风景：

 

　　1.　由于原始的照片文件位于外部储存设备上，现在要“灌”到电脑中，须指定照片文件的存放位置．这些位置可以是在本地建立的文件夹，也可在网络上指定某个位置．这个对话框一共有三项选择：

　　● 照片命名．

　　windows xp虽然会自动提供缺省的照片文件名称，但缺省名称是Picture打头，后面紧跟是数字，圆点后面是扩展名，例如Picture120000.jepg，毫无提示作用。不如你给照片取一个唯一的名称。

　　● 选择照片文件存放的目的地文件夹

　　处理好的照片文件将存放在这个文件夹中．你可以手工输入文件夹名称指定，也以点击Browse...来指定．

　　● 单选：在拷贝到计算机上的同时，是否还将这些照片拷贝到Internet上？

　　可选Yes 或No．如果你选了Yes,会要求你指定网络的具体位置．

　　最后，有一复选框：当将照片传送到计算机中之后，是否删除原来保存在数码照相机中的（原始）照片？你根据自己的要求进行选择既可．选择好了目的地，就请各位快来看下面：

 
 
 
    2.　拷贝照片：

　　接上一步，向导帮你从选定的目录中直接进行照片的拷贝，对话可框显示出位置、照片名称、本张照片完成拷贝的百分比、总共需要拷贝的照片张数以及已经完成的张数．从图中可以看出来，默认情况下照片名称使用的是数字．这样对于按日期或特殊要求检索显得不方便，不能不说是一个小小的不足．

　　到此为止，我们已经走完了旅途的第一阶段．学会了如何把外部的照片文件输进电脑中！

　　第二景点：使用数码照片显示探测器

　　这一站将是本次旅程中景色最好的部分：向导会给我们展示多种多样的照片显示方式，使你大饱眼福．
○　Windows Explorer与EXIF联手支持的智能显示大家都很熟悉微软的浏览器IE，是：Internet Explorer的缩写．中文意思是：网络探险家．在windows xp中，微软又叫来了一位探险家，名字为Windows Explorer，对这位．我们有必要多介绍几句：Windows Explorer可以探测显示对象的许多参数，然后以最合适的方式显示对象，比如，在浏览器中仅仅显示一幅图片时，早期的浏览器窗口大小和图片大小不会自动协调到双方一致．有了Windows Explorer，就可以总保持两者尺寸一致．当然，这位新来的探险家，不会就这么一招功夫．它还可在EXIF信息的支持下，对数码照片提供许多不同效果、不同方式的显示． EXIF是Exchangeable Image File的缩写，中文含义为"可交换图标文件"．是日本电子工业振兴协会制定的一种文件格式．其详细介绍可以参考有关资料．有Windows Explorer和EXIF联手，在照片向导的协助之下，可以做到对照片的详细显示、简略显示、平铺显示、幻灯片显示等多种模式．且使用起来易如反掌。

　　好了！我们先介绍详细模式：

 

　　详细模式对话框窗口，被明显地分成左右两个部分：左边是任务作业区．可以选定当前允许执行的照片操作任务，文件操作任务，以及其它位置的任务．左下角是当前选定的照片的详细情况．具体可以参看上图．

　　照片任务中:

　　View as a slide show:按幻灯片方式显示．
　　Order prints from the Internet: 网络订购打印照片．
　　Print Pictureds : 打印（本地）照片
　　Set as desktop background: 将选定的照片作为桌面背景图片．

　　文件任务中：
　　Rename this file : 将选定文件重新命名．
　　Move this file: 移动这个文件．
　　Copy this file:　拷贝这个文件．
　　Publish this file to the Web:　将这个文件发送到Web页面．
　　E-Mail this file:　　将这个文件作为邮件发出． 　
　　Throw away this file:　丢弃这个文件．　
　　以上这些，就好比是这个景点的分布简图　

　　对话框中间区域的右边是关于照片文件的详细信息．这些信息有：

　　照片文件名称、文件大小、拍摄的日期与时间、照片长和宽、数字相机使用的模式等等．

    为了理解下一步骤，请特别注意上面窗口中的thumbs.db文件！这个文件只有大小，没有其它信息．它的保存的内容实际上也不是照片．而是一个管理文件．thumbs.db文件是简略模式显示所需要的．甚至可以说是一个相当不错的工具，比之windows2000作了较大的改动．不过，仍然不兼容那个很有名气的看图软件：ACDSEE,这不能不说又是一个小小的遗憾．

　　我们再看简略模式的情况：窗口还是那个窗口，但右边显示的不再是枯燥的文件信息，而是照片的缩略图．仔细看看，会发现和详细模式窗口中的文件是一一对应的．如此奇妙的效果，正是thumbs.db文件的功劳！

 

　　该窗口还可以支持传统的平铺显示模式，像下面那样：

 

　　像上图这种画面，我们在早期的windows中似曾见过，如果说有变化，也仅仅是更美观些罢了．

　　另一种显示模式是所谓的幻灯片显示，大概是因下面的一排照片很像摆开的幻灯片或伸展开的电影胶片吧？所以有这么个名称。这种模式对照片处理有较大的使用价值．请看下图，　请特别注意窗口中大幅图片下面的一排按钮：　


    这些按钮的功能理解起来很容易，有：放大，缩小，某一方向的尺寸调整，旋转以及作自然的卷动效果．实际上，上述窗口就是实现了直接对照片的预览．

　　上述的操作强调视觉效果，对于排版、预览等很相宜．但对于关心技术细节的人，更希望得到具体的数据。好比旅游者中，有对山川风物钟情，有对碑林记述留意，各取所需。windows xp也支持照片的技术参数显示：你选定某一幅照片后，就可以显示出其详细的数据信息，下面窗口中就是上图中大楼照片的详细地数据信息：　　　　

 

　
　　这些数据是在照片拍摄时，数码相机记录下来的各种参数，很多是专业性很强的知识，例如，镜头的焦距，光圈值，景深、拍摄的日期时间等等．有些无法提取的参数：例如快门速度等等，是没有记录的．由于我在这上面的水平低，能力不逮．你真欲弄清楚确切含义，敬请参考有关资料。
