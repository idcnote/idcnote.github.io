1．什么是微软Windows Mobile 5.0?
最新的Windows Mobile 5.0不叫Windows Mobile 5.0 2005，而是Windows Mobile 5.0。此次微软推出最新版的Windows Mobile 5.0智能型手机或PDA的操作系统，虽然只有一般手机的大小与重量，却充分整合微软Office的各项功能；Windows Mobile 5.0 不仅具有MSN、收发email、行事历、通讯簿等各能…等更多酷炫的功能。并具备永续性的内存储存，使数据管理上更有效率；而功能强大的多媒体Windows Media Player 10 Mobile，并且支持硬盘。使用者可享受与使用这些各式各样的功能与服务。
忙碌的上班族希望拥有一个兼具手机通讯、PDA Outlook、PC文书处理、无线上网的”全功能计算机手机”。 每天辛苦工作的上班族，经常在外劳碌奔波，而手机、PDA是随身必备的生财工具，而有时还要背着厚重的 Notebook跑来跑去。『行动工作族』不是ㄧ个特殊行业的专有名词，而是每个上班族的普遍身分，无论你是快递货运司机、医院里的医生护士，亦或是每天在外业务人员、会议不断的高阶主管…等，如何让工作更有效率、更节省时间、实时存取信息，即成为ㄧ个不可忽视的课题。

2. Windows Mobile 5.0智能型手机差异为何 ?
Windows Mobile 5.0 的 Pocket PC Phone Edition 除结合了标准 Pocket PC 的全部功能外，还加入了全面的电话话音与发送简讯功能。Pocket PC Phone Edition 的 Windows Mobile 5.0已将PDA 和手机的使用经验合二为一，用户可根据手机上的连络人清单拨号、收发 SMS 短讯，以及记录来电人的身分，并且透过无线网络可随时随地收发电子邮件。
Windows Mobile 5.0 的 Smartphone 多数的功能与Pocket PC Phone Edition一样，但将 PDA 所具备的功能整合到跟现有手机相同。但 Smartphone 强调符合单手操作之用的袖珍数字键盘，与Pocket PC Phone强调以触控屏幕方式操作的方式不同。

4. 目前使用的 Windows Mobile 5.0版本为何 ?
目前采用的是Windows Mobile 5.0，此版本是按行动装置包括Smartphone 、 Pocket PC、Pocket PC Phone等特别设计的Windows CE 5.0版本。

5. Windows Mobile 5.0 有什么新功能 ?
有关Windows Mobile 5.0主要特色包括︰
a. 新增“永续储存”内存，无电情况下也可以保持所有资料
b. 新版办公软件套装：Word Mobile, PowerPoint Mobile Viewer以及Excel Mobile
c. 使用新版ActiveSync 4.0
d. 强化了智能型手机对WiFi的支持
e. 加入支持Push-To-Talk和视讯电话功能
f. 加强了蓝牙传输支持
g. 重新编写的数字相机镜头的应用程序接口，软件开发者为内置摄像头机型开发应用软件将更为简单。
h. 支持USB 2.0，使用Windows Mobile 5.0系统的机型可直接与外部移动硬盘连接传输

6. Windows Mobile 5.0 新平台是否更具弹性?
Windows Mobile 5.0智能型手机的平台具备更高的弹性，满足特殊的客制化需求，使合作伙伴厂商可轻易地加入差异化的技术，例如 “即按即说”(push-to-talk)或VOIP视讯通话与会议。

7. Windows Mobile 5.0 是否可单手操控?
Windows Mobil 5.0 平台可整合软性按键(Soft-key)，具备水平或垂直显示之旋转变换，支持QWERTY键盘。因此合作厂商能够开发出极佳的装置，并以单手操控键盘，让使用者可以不需触控笔即可操作产品取得信息。
Windows Mobile 5.0是将PDA的功能与行动电话整合的行动装置。Windows Mobile 5.0是设计为一手即可操作语音和数据数据功能。这可应用于声音和文字通信； 透过无线网络联机存取公司与因特网的加密数据与服务。 Windows Mobile 5.0提供你一个在移动时透过语音和文字存取各式数据和服务的选择。

8. Windows Mobile 5.0 是否支持高分辨率的触控屏幕 ?
Windows Mobile 5.0 的触控屏幕，除原有320x240的分辨率外，部份机种甚至有640x840支持高分辨率的液晶显示屏幕。液晶显示屏幕（LCD）是一种适用于计算机、Pocket PC 及任何流动装置设备的显示屏幕技术。薄膜晶体管（TFT）显示屏幕上每一个像素（pixel）都用上一个独立的晶体管的 LCD 面板。这种技术让所有 Pocket PC，可以在较低耗电量前题下，提供显示反应高速，色彩又亮丽的视像。个别型号甚至采用具备背逆光背景合成式 TFT 显示屏幕，以便在光线昏暗的环境下，仍提供最大的可视度。

9. Windows Mobile 5.0是否可变换直向与行横向显示操作 ?
Windows Mobile 5.0 Pocket PC 与Pocket PC Phone的屏幕显示，除了直向的屏幕显示外，还横向屏幕显示的选择。由于横向接口的显示，类似PC的屏幕显示，也较利于网络浏览的习惯。但除了原有Windows Mobile 5.0的内建操作系统与程序外，多数外挂的应用程序不支持横向显示功能。此外，Windows Mobile 5.0 Smartphone的操作系统不支持此横向显示功能。

10. 如何使用手机播音功能 ?
如果您想使用Windows Mobile 5.0 的手机播音功能，需先选择播出的联络人电话或自行输入电话后，按下电话播出电话大约4秒后，即可自动转为手机播音功能来使用。每次使用后此功能均会自动结束。

11. 在 Windows Mobile 5.0上 MP3是否可做为铃声 ?
MP3在新版本是支持MP3铃声，而提醒铃声大小或种类，可至”设定”" ”个人” ( ”声音”，来设定或选取。可依行事历、电子邮件、来电提示….等选择各式预设铃声或MP3音乐文件，来做区别。

12. 如何选择 Windows Mobile 5.0铃声 ?
Windows Mobile 5.0装置可自由选择各种情况所使用的铃声模式，包括来电铃声、收到邮件、简讯、会议通知等，可由使用者自订铃声，以便区别，而Windows Mobile 5.0装置支持.wav , .mid和.wma(Windows Media Audio)档，需特别注意当铃声档案越大，将再来电时花更长时间去启动播放。

13. 如何存放想要铃声的声音档案 ?
Windows Mobile 5.0装置所使用的铃声档案应该放到装置中的\Stoage(储存)\Application Data(应用程序数据)\Sounds(声音)目录中。当您想要选取已经复制到装置上的铃声，则应选择铃声，并至'Start(开始)'à'Settings(设定)'à'Sounds(声音)'，并选择你已复制到装置中的档案来当作铃声。

14. Windows Mobile 5.0 如何储存已发送的讯息?
要保留一个已发送讯息的复本，从’Start(开始)’清单(打开’Messaging(讯息)’应用程序(按下’Menu(清单)’键(再按下’Options(选项)’和’Sending(寄送)’ (检查’Save copies of sent item(储存已寄送项目的复本)’文件匣，并按下’Done(完成)’

15. Windows Mobile 5.0 可在讯息中编辑’My Text(我的文件)
‘My Text(我的文件)’提供了一种简易地增加预先定义的文字到寄送讯息内容的方式，想要编辑在’My Text(我的文件)’中出现的句子，可从’Start(开始)’清单中(打开’Messaging(讯息)’应用程序(按下’Menu(清单)’键(再按下’Options(选项)’和’Edit My Text(编辑我的文件)’。从这里]，你可以更改可以预设使用的文字句子。

16. 如何在我的 SIM卡上建立联络人纪录( Smartphone / Pocket PC Phone only)
当Windows Mobile 5.0中的Pocket PC 与　Smartphone 的智能型手机，在SIM卡上的联络人，每个联络人纪录只可以有一个号码。然而，如果你希望在SIM卡上建立联络人，你可能要在联络人应用程序中，藉由选取’Menu(清单)’键à选择’New SIM contact(新的SIM卡联络人)’ 来达成( 在Smartphone找不到)。

17. SIM 通讯簿与 Outlook联络人有何不同 ?
Windows Mobile 5.0中的SIM 通讯簿与Outlook联络人不同，但联络人会显示装置上全部’Contacts(联络人)与SIM通讯簿中所有的数据与电话。SIM 通讯簿就是储存在SIM卡上的通讯簿数据(只有GSM)。这与你存在装置上的通讯簿不同。且SIM通讯簿只能存一个号码且无法同步。( Smartphone / Pocket PC Phone only)

18. Windows Mobile 5.0是否具备电显示功能
当Windows Mobile 5.0中的Pocket PC 与　Smartphone 的智能型手机，具备来电显示，的功能，可设定阻文件您不想接听的来电。( Smartphone / Pocket PC Phone only)

19. Windows Mobile 5.0如何增加电子邮件账号 ?
Windows Mobile 5.0 如增加电子邮件账号,由’Start’开始清单打开à’Messaging’讯息中心的应用程序，并选去下’Menu’键，再按下’Options’和’Account Setup，再由此画面来选择’Menu’键和’New’，即可开始建立一个新的电子邮件账号了。Windows Mobile 5.0 支持最多达八个讯息账号，这些讯息账号包括ActiveSync,POP3或IMAP4,Text Messaging(文字讯息)以及’Multimedia Messaging’(多媒体讯息)账号。

20. Windows Mobile 5.0如何设定电子邮件签名 ?
使用者可以藉由使用电子邮件签名，并自动地增加文字到任何所寄送的邮件中，当你要附加你的联络讯息或其它标题到你的外送邮件时。这将会为你开创了更方便的方式，而不管使用的外送账号是什么；一个不同的签名可以被任一个账号或是应用到所有外送邮件的账号所使用，要建立一个新签名，从’Start(开始)’清单中à打开’Messaging(讯息)’应用程序, (按下’Menu(清单)’键(再按下’Options(选项)’和’Signatures(签名)’。此后，你可以选取所建立的签名，并选择账号后插入任一文字邮件中。

21. 如何使用 Windows Mobile 5.0发出实时讯息 ?
你可以借助 Pocket PC、Pocket PC Phone Edition 或 Smartphone 内建的 MSN Messenger，享有如桌面计算机同一程序所提供聊天环境。程序具备自动登入、查看在线联机人清单、实时讯息收发，以及同步与多个连络人实时对话等功能。但注意如要使用实时讯息收发功能，你的 Windows Mobile 5.0 装置必须透过 Wi-Fi 或手机流动网络接驳上网。

22. Windows Mobile 5.0如何发送 SMS/MMS简讯 ?
先至收件夹，选择SMS简讯发送，并新增一封简讯，如为SMS内文在160 Bytes以内，或为MMS为添加图片；再直接由联络人清单中选取寄件者，透过电信公司SIM卡直接发出简讯。

23. 如何在 Windows Mobile 5.0 使用 MSN?
使用Windows Mobile 5.0 MSN功能相当简单，需先无线上网完成联机，再选取MSN登入个人PC上MSN上相同的账号密码即可，所有的群组与联络人无须重新输入，将自动同步至用Windows Mobile 5.0 MSN上。然而目前Windows Mobile 5.0 中的MSN并不具备电子白板、语音、档案传送、网络摄影机…等PC上MSN的功能。

24. Windows Mobile 5.0 是否加入支持视讯电话功能?

38. Windows Mobile 5.0暂存内存架构为何?

41. Windows Mobile 5.0 是否具备永续储存的功能?
Windows Mobile 5.0有一项非常新的特性：持续的存储能力，当失去电力后数据不会消失。这样即使大家不小心把电全部耗光，PPC里面的数据仍然存在，不会损失。为此微软采取了和Windows Mobile 2003版本中完全不同的内存管理方法。
事实上，PPC上的内存也就两种，ROM和RAM（内存），在现在的WM2003的机器中，内存同时存储应用程序和文件；而ROM则只是存储系统的，ROM剩下的空间也可以通过软件刷进去一些程序。
过去2003版本为了使得内存一直保存数据，电池必须一直供电，一断电数据就全部丢失了，所以说即使你关闭机器后，电池仍然在给内存供电以保持里面的数据不丢失。所以PPC不可以像计算机那样有着非常大的内存，比如512MB，那样也许一个晚上的待机就会把所有的电力都耗尽了。而ROM里面的数据就像我们的计算机硬盘那样，即使掉电后依然能够存在。
其实新的Windows Mobile 5.05.0的系统电全部耗光时，内存数据就是依靠ROM的掉电不掉数据的，和上文所说的内存和ROM的使用情况不同，WM5.0只是在内存中存储一些正在运行的程序，而应用程序以及数据都是存储在ROM里面的。这样就可以像计算机一样，断电后再开机资料依然没有丢失。
同时，由于数据不在保持的在内存中，就不需要持续的电力来维持内存中的数据，我们也就有可能在PPC上安装更多的内存来运行程序。不过同时也有很多人指出，一切都是从ROM中读取数据会降低PPC的速度的，其实现在的智慧手机已经采取这样的措施了，也并不是非常的缓慢

如果你手动地按下’Sync(同步)’键，但是这选项和你的装置却未被连接到使用ActiveSync的个人计算机时，这装置将会试着连接到你已建立伙伴关系的个人计算机。如果你的装置并没有连接到那台个人计算机，这将需要一个VPN(虚拟私人网络)或是RAS(拨接连线)至贵公司或私人网络.如果VPN(虚拟私人网络)或是RAS(拨接连线)设定并未存在，你将要视查在装置上ActiveSync(自动化同步)画面是否有错误。

95. Windows Mobile 5.0可否与Apple Mac系统同步?
ActiveSync并不支持Macs的联络人或行事历的同步功能，但email.可适用 POP or IMAP 与Windows Mobile 5.0 行动装置进行由邮件传送。

96. Windows Mobile 5.0可否与Lotus Notes系统同步?
ActiveSync并不支持与Lotus Notes的联络人或行事历的同步功能，但可选择策略厂商提供的PC前端或Server产品同步联络人或行事历数据，亦或仅以 POP or IMAP 与Windows Mobile 5.0 行动装置进行由邮件email.传送。

97. Windows Mobile 5.0支持哪些扩充槽?
Windows Mobile 5.0有些机种支持SD或MMC卡。SD 是一种体积相当于邮票大小的闪存卡，为数据、应用程序和音乐文件提供外部存储器，在 SD 上保存的数据可以轻易在流动装置和桌面计算机间传递。
MMC这是一种体积仅相当于邮票大小的可移动式记忆卡。MMC 可用来储存数据，例如：图像、影像或文字等数据，并能够在 Pocket PC 与桌面计算机间作数据交换。MMC 的外形跟 SD 十分相似，而主要分别在于 MMC 缺少 SD 所具备的数据防止写入开关。

98. 当Windows Mobile 5.0内存不足时?
因为Windows Mobile 5.0从来不真正关闭，开启运行的程序一直不关闭。所以当您开启太多运用程序来执行时， 虚拟内存会产生分配不足的现象，当内存少的情况下，企图增加堆栈的程会失败，而内存漏失逐渐积聚并会 严重影响性能。此时您可以到"设定"- -“系统"-- “内存"的选项下，选择"执行程序" 将目前暂不需要的应用程序给"停止"，便能释放出部分内存使其正常运作。如果程序在8秒内没有关闭，系统将强制终止它的程序并把它从内存清除，这可能导致大量数据丢失。

99. Memory内存容量如何升级?
Smartphone的标准内存容量为16MB，而Pocket PC的标准内存容量为64/128 MB。使用者可依所购买的Windows Mobile 5.0 行动装置所具备的SD /CF/MMC插槽格式，来选取兼容的Storage Card的扩充套件，记忆卡扩充内存的容量到64 MB, 128MB, 256MB, 526 MB ..1G

100. 如何结束正在执行的程序?
刚接触Windows Mobile 5.0的人在执行许多软件时，常会发现没有Exit这个选项，而无法结束程序。虽然Microsoft宣称并不需要结束任何程序，Windows Mobile 5.0自己会管理这些任务。但还是常有许多旧版的软件在Windows Mobile 5.0上无法顺利执行，此时若想结束他就必选择 "设定/系统/内存/执行程序"才能结束此程序，实在不方便。可直接点选右上方"OK"键关闭，或执行小键盘，按ctl后再按q即可结束程序。

101. 至何处寻找下载Windows Mobile 5.0应用程序?
使用者可至Windows Mobile 5.0 Community网站中的Mobile Application Catalog搜寻选取Pocket PC 或 Smartphone的各式应用程序。亦可至Handango Software Store http://www.handango.com 网站来查询或采购应用程序软件。

102. 如何将应用程序安装于Windows Mobile 5.0?
当使用者选取并下载完成Pocket PC 或 Smartphone的各式应用程序于PC/NB计算机中，先需先建立同步联机，并点选免费或采购应用程序软件，应用程序将直接同步上传至Pocket PC 或 Smartphone。但在联机之前需先确认Windows Mobile 5.0中内存是足够的。

103. 如何移除Windows Mobile 5.0中的应用程序?
应用程序可在你的Windows Mobile 5.0装置连接到PC时ActiveSync中的"新增/移除程序"移除，或者直接从设备移除。 由ActiveSync移除，从工具选单中选择新增/移除，取消选择你希望移除的应用程序。或者直接于Windows Mobile 5.0装置中，"开始"à"设定" à"移除程序"，再选择你希望移除的应用程序。

104. 什么样类型的应用程序不能与Windows Mobile 5.0兼容?
大多数为Windows Mobile 5.0设计的应用程序，应该可以在设备上运作。但是 由于硬件差别可能会稍微地改变每个Windows Mobile 5.0设备，相同的程序在不同的设备上可能会有不同的状况。然而因为Smartphone操作系统与接口稍有不同，多数支持Pocket PC或Pocket PC Phone的应用程序，可能会有另ㄧ个支持Smartphone的版本。另外， 一些行动装置的操作者和设备制造商可能会进行限制以防止未受测试或没有支持计划的应用程序以保护你的设备。

105. 选择安装应用程序的储存位置?
当有储存卡在你的设备上，安装应用程序时，你可能会被提醒要求安装应用 程序的位置。 安装在储存卡上相对于安装在内建内存装，可以保存更多设备内的储存空间并且可以允许你安装更多的程序。 另外， 在记忆卡上安装应用程序将允许你使用空间储存的电子邮件，通讯簿，日历和其它数据。 如果你选择安装在储存卡上，如果你从设备把这张卡片拿走，应用程序将无法执行。

106. Windows Mobile 5.0是否内建计算器功能?
Windows Mobile 5.0内建计算器功能，让您不用在携带另外一台计算器。当您出国计算汇率，计算订单价格时均非常方便。

107. Windows Mobile 5.0是否可外挂电子地图与汽车导航?
PDA电子地图与GPS导航系统，是现今最热门的应用，电子地图全台最各地的地图数据库、全省高速道路、全省道路数据等，只要安装GIS电子地图软件，使用者即可方便的查询全省的地图数据。此外，有部份厂商甚至提供国外GIS电子地来下载试用。而结合GPS全球卫星定位，更可透过 Windows Mobile 5.0提供语音的汽车导航服务，包括最佳路径规划、智能语音导航、全省测速照相提醒等。

108. Windows Mobile 5.0备份联系人最简单有效的方法：
打开《资源管理器》，在《我的设备》中复制pim文件，备份到SD卡或者电脑上。恢复时将《我的设备》中pim文件先重命名随便增加或减少一个字母（不然无法将同名的pim备份文件替换现有的pim文件），再将备份的pim文件粘贴后，重启后即恢复备份时原有的联系人。随后再在《我的设备》中将恢复前重命名的原pim文件删除。
优点：在系统硬启清空后及时恢复联系人，免除非要系统整体备份或电脑outlook同步来恢复联系人的麻烦，还可将同一系统的联系人相互复制。
