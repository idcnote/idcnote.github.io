文章正文

导语：随着Windows 8消费者预览版（即Beta版）发布日期的确定，关于Windows 8的新闻和评论也多起来。距离发布Windows 8消费者预览版仍然还有一段时间，甚至即将发布的Windows 8 Beta版内部版本号是8225还是8250仍还未知。为了更好的让大家了解Windows 8，笔者特地撰写此文，分析Windows 8怎么样？其未来究竟如何？

上周微软发布消息，确定Windows 8消费者预览版将于2月29日在巴塞罗那召开的世界移动通信大会上发布并提供下载，根据之前开发者预览版以及泄露出来的各种编译版本，大家基本上都对 Windows 8或多或少有点了解。笔者去年曾安装Windows 8开发者预览版，随后也密切关注其新版本的最新特性以及改进，客观的分析：Windows 8可能是一个不错的操作系统，在平板电脑系统份额能占一席地位，但对终端PC消费者而言，也许就如同鸡肋一样：食之无味，弃之可惜。

Windows 8都有什么？

月底Windows 8 Beta发布后，其大框架基本就确定了，在随后的RC版、以及最终的RTM版本中，一般不会再有功能性更新，主要是对Beta版本中所产生的问题进行修补，对其兼容性、以及系统进行优化。根据微软官方和用户的评测，Windows 8新特性主要有以下几个方面：

  • 启动速度加快：据说Windows 8能够在十秒左右迅速完成启动。另外唤醒睡眠的时间相对Windows 7也有相应的提高。
  •  Refresh & Reset：类似于手机上的重置和刷新功能，Reset（重置），恢复至默认设置，会丢失所有的数据和应用程序。Refresh（刷新），恢复系统至默认状态，但保留原有的数据和应用程序。
  • Windows to GO：可以将系统装在U盘上，方便IT人员随身带着自己的系统。
  • Metro界面：专为触摸屏电脑和平板电脑设计的用户界面。
  • Windows 应用商店：内置的Windows应用商店，方便用户搜索和购买微软认证的应用程序。
  • WiFi Direct支持：Wi-Fi Direct可以为我们提供随手可及的近距离的点对点共享。
  • 支持本地ISO镜像：可以在Windows 8系统下直接对ISO镜像文件进行复制、查看、粘贴等操作。
  • Hyper-V：企业通过Hyper-V，可以运行其他的操作系统，包括旧版的Windows。
  • Ribbon式资源管理器：Windows 8的资源管理器新增了类似Office中使用的“Ribbon UI”，它可以根据不同类型文件提供更多的操作按钮图标。
  • 全新的任务管理器：Windows8将自带和系统更加协调的任务管理器UI以及更强大的程序管理机制，可以给用户带来方便。
  • 其他：比如支持USB3.0、Aero支持CPU运算等。

为什么食之无味 弃之可惜？

1.没有功能性吸引：相对于以前Windows系统受到用户热捧，则更多的是时代的烙印。比如九十年代初的 Windows 3.1首先增加了声音输入输出最基本的多媒体支持，让人们看到了PC机娱乐的前景。而Windows 95基本定型的视窗界面，不仅让消费者获得视觉冲击，更让操作变得简单和更容易接受，对用户而言诱惑力极大。其后的Windows 98、2000以及XP，每一次的升级都蕴含了巨大的变革和突破，尤其是Windows 2000，全新的内核融合了DOS和NT，使得Windows 2000成为一个独占式、可中断执行、极其稳定和安全的操作系统，而Windows XP则把基于文本、图像、动画、声音、视频的多媒体技术发挥的淋漓尽致，让用户无法抗拒如此魅力四射的系统。

可以说每次系统的更新换代，都有巨大的技术推动和革新，相对于Windows 7效率的内存管理、交互技术的相当完善，用户没有理由也不会迅速升级到Windows 8。因为Windows 8没有什么新功能可以吸引用户的。支持USB3.0？今年年底推出的Windows 7 SP2也同样支持。而且Windows 8新增本地直接修改ISO镜像文件又有多少人能够用到？何况有些免费软件比之还更好。谈到启动速度，优化后的Windows 7速度也不慢。新推出的Windows 应用商店，从本质上来看是为平板电脑准备的，因为X86版所积累的软件可以用超多来形容。而Refresh & Reset功能，实际上Windows 7里也有，只不过步骤复杂不好操作而已。被人称到的Windows to Go功能实际上早有，也就是我们提取Windows内核后的Windows PE，只不过微软将这功能官方化了而已。

2．个人习惯：被大家谈论最多的或许就是Windows 8 Metro界面，但Metro界面带给人的感觉更多的像是一层皮，当功能超过这个皮功能的时候，就会露出最原本的样子，另外Metro界面也让一些用户无所适从，在没有触摸体验的PC上甚至可以说是令人讨厌的东西。Ribbon式资源管理器虽然可以折叠，但是展开后占用了太多的显示空间。

3.适合即最好：现在的消费者越来越理性，不再像之前那样盲目更新自己的操作系统，因为他们认识到软件以及系统并不是最新就最好，一款系统重要的是稳定、够用和适合自己。尤其是Windows XP的成功和Windows Vsita硬件要求高、程序不兼容，导致六年之后应该在Vista时代升级的电脑，都拖到Windows 7时代集中升级，相对而言，Windows 7是新平台，使用也还满意，因此很多人不会再耗费金钱或者改变使用习惯升级到Windows 8。对于大部分升级到Windwos 7的企业用户来说，企业办公讲究的是效率、实用，不可能代代都换系统，所以也不会再去耗费经费升级到Windows 8。

笔者觉得Windows 8推出后会有大量用户安装试用，但新鲜、好奇期过去后，用户就会在想：因为习惯，因为Windows 7的完善，因为届时打了SP2补丁包的Windows 7稳定……，最后是不是要换回Windows 7？

分析：并非一无是处 Windows 8平板或异军突起

当2010年iOS引入平板电脑，让平板电脑真正成为一种带动巨大市场需求的产品。平板电脑取得的成功以及微软的Windows CE以及Windows Phone 7市场份额迅速萎缩，让微软认识到在便携设备上的不足。笔者认为，Windows 8的推出似乎更侧重于平板电脑等便携设备。对于桌面系统而言Windows 8更像是过渡产品，其命运类似于Windows ME，重点可能是在下一代操作系统Windows 9上。

相比较而言，Android平板在微软真空期内并没获得像iPad那样的成功，更重要的是Android平板不稳定，以及经常无法快速完成渲染网页。另外Google升级Android太快，导致每次改动过大，给开发者和厂商带来压力。同时Android系统程序不自动结束、浪费资源、浪费电量以及安全性等问题也不断给人诟病。而Windows 8对USB设备的支持、Office办公、丰富的软件、以及强大的娱乐性，同时微软结盟的厂商大力推广，或许Windos 8平板会迅速流行起来。

最后：至于Windows 8具体怎么样，那就仁者见仁智者见智了。笔者仅仅希望通过此文能够抛砖引玉，吸引网友踊跃在评论中阐述自己观点。
