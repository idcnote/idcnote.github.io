Vista操作系统使用疑问解答_操作系统

内容摘要
Windows Vista操作系统已经发布了很多个版本的测试版了，相信大家很多人都已经试用过了，笔者给大家整理了一些Vista的相关疑问。
文章正文

    Windows Vista操作系统已经发布了很多个版本的测试版了，相信大家很多人都已经试用过了，笔者给大家整理了一些Vista的相关疑问。有不正当之处请指正。

　　问:你好，请问在Vista中怎样删除已激活的当前用户帐号?

　　答:在电脑上点右键，选择管理，在本地用户和组里选用户，在你建立的账号上点右键选删除，记着要在Administrator账户上点右键，选属性，Enable它，确定，注销就可以用Administrator用户登录。这样就删除了已激活的当前用户帐号。

　　问:Windows的新系统Vista要怎样的硬件支持?

　　答:针对打算通过Windows Vista体验更精彩数字娱乐的进阶用户，微软(Microsoft)提供了第二级别最低PC硬件配置清单。细节内容如下:

　　·中央处理器——主频为1GHz的x86/x64处理器

　　·系统内存——1GB

　　·图形处理器——能够支持Windows Aero用户界面效果的图形处理器

　　·显卡显存——128MB

　　·硬盘空间——40GB

　　·光存储外设——DVD-ROM

　　·音频——Windows兼容的音频输出

　　·网络接入——Internet接入

　　问:Win Vista 到现在有几个版本?

　　答:Windows Vista Starter Edition:它是一款32位操作系统，主要面向新兴市场的入门计算机用户，只支持同时运行3个应用程序，不支持登录密码或者快速用户切换(FUS)。总体而言，Windows Vista Starter Edition类似于Windows XP Starter Edition。

　　- Windows Vista Home Basic Edition:它是Windows Vista的简易版本，主要面向单PC家庭用户。Windows Vista Home Basic Edition是其它Windows Vista版本的基础，支持Windows防火墙、Windows安全中心、安全无线网络、防垃圾邮件/病毒/间谍软件、网络地图、Windows搜索、Aero用户界面、电影制作、图片库、Windows媒体播放器、带有RSS支持的Outlook Express以及P2P信息工具等功能。总体而言，Windows Vista Home Basic Edition类似于Windows XP Home Edition。

　　- Windows Vista Home Premium Edition:它在Windows Vista Home Basic Edition的基础上，加入了全部的家庭娱乐和个人制作功能，主要面向PC发烧友、多PC用户和笔记本电脑用户。除基本功能外，Windows Vista Home Premium Edition还具有媒体中心和媒体中心扩展功能、DVD视频编缉、HDTV播放、Tablet PC以及Wi-Fi音频配置和漫游等功能。总体而言，Windows Vista Home Premium Edition类似于Windows XP Media Center Edition。

　　- Windows Vista Professional Edition:它主要面向各种规模的企业，加入了域名加入和管理功能，同非微软网络协议(如Netware和SNMP)兼容，具有远程桌面、IIS Web服务器和加密文件系统等功能。总体而言，Windows Vista Professional Edition类似于Windows XP Pro Edition。

　　-Windows Vista Small Business Edition:它是Windows Vista Professional标准版的扩展，主要面向小企业，包含了备份、PC传真和扫描等特殊功能。Windows Vista Small Business Edition可以升级为Windows Vista Enterprise Edition或者Windows Vista Ultimate Edition。微软并没有推出面向小企业的Windows XP小企业版。

　　-Windows Vista Enterprise Edition:它是Windows Vista Professional标准版的扩展，主要面向企业决策者、IT经理和员工，包含了虚拟PC、多语言用户界面、完全加密技术等高级功能。微软并没有推出类似版本的Windows XP。

　　-Windows Vista Ultimate Edition:它是微软迄今为止推出的最佳PC操作系统，具有Windows Vista Home Premium和Windows Vista Pro Edition的所有功能。此外，它还加入了特殊的游戏功能、播客功能以及网络俱乐部等功能。微软计划向Windows Vista Ultimate Edition用户提供扩展A1服务、免费音乐下载、免费电影下载、网络娱乐软件和优先支持等服务。目前，Windows Vista Ultimate Edition的产品定位还不清楚，微软并没有推出类似版本的Windows XP。

    问:你好，我安装了Vista 5472版，显卡是FX5900。装了那个88.61的beta显示驱动后找不到显卡的控制面板，就根没装驱动一样。请问该怎么办?

　　答:现在NVIDIA显卡的控制面板集成在系统的控制面板里，进入系统的控制面板就可以找到了。

 

　　问:Windows Vista ADSL拨号连接怎么设置

　　答:Windows Vista下拨号连接设置和Windows XP是一样的，打开网上邻居，在左边的选择栏里面选择“创建一个新的链接”。然后再按照要求一步步输入相应帐号和密码就可以了。

　　问:怎样激活Vista

　　答:目前没有真正的激活Vista的方法，只能去除14天的使用限制!具体方法如下:

　　1.运行services.msc，打开服务管理器。

　　2.禁用Software Licensing Service服务，重启

　　到5342为止，此方法可行，后续版本测试后会更新该方法可行性!

　　问:本人用的是Athlon64 3000+，C51G主板，1G内存。现使用内置的6100显示芯片，共享256M内存作为显存，装完Vista 5472后一切正常，可以看到半透明特效，最大最小化窗口时还有特别的动画效果。再弄好网络上网装了个Flash Player 9，回头又装了主板驱程，包括显示驱程和声卡驱程。不知什么时候透明特效和动画效果就没了，怎么搞都没办法实现效果了，不知怎样才能打开特效?而且卸载显示驱程后分辨率和刷新率都跟2000没装驱程时一样，非得再装回显示驱程才正常，为什么新装的系统不用装显示驱程都可以透明，装了驱程后反而不可用了呢?

　　答:问题就在于你安装了显卡驱动，你安装的是for XP的驱动程序。使用Vista 自带的驱动程序就可以了，不能使用以前for XP的驱动程序。现在你要做的只能是安装Nvidia 提供的 for Vista 驱动了:http://cn.nvidia.com/object/winvista_x86_88.61.html 。

　　问:如何让Vista支持中文软件?

　　答:需要通过修改区域和语音选项，方法如下:(从5308版本开始不再需要，仅需在第一次进入桌面前选择)

　　1.打开控制面板，然后点击“Change the system language”，打开“Regional and Language Options”对话框，选择“Administrative”选项卡，在“Select a language for non-Unicode programs”下拉菜单中选择“Chinese(PRC)”。

　　2.打开“Regional and Language Options”对话框的“Formats”选项卡，在“Current Formats”下拉菜单中选择“Chinese(PRC)”.

　　3.确定重启即可!

　　答:Strat--Network--点击上方的Netwoek Center--左边侧栏的View connections

　　问:谁知道Vista的DX在什么地方可以找到?

　　答:运行里(windows key + R) 输入 dxdiag

　　问:不支持DX9的显卡，如何开启DWM效果?

　　答:请先想办法安装XP的显卡驱动(例如用兼容性模式安装等)，然后使用以下程序开启!

　　若安装SP1后想关闭，请将注册表HKEY_CURRENT_USER\Software\Microsoft\DWM "UseMachineCheck"=dword:00000000的键值改成1或者删除!

　　注意:部分显卡开启DWM后可能造成系统速度下降或无法进入系统等不良后果!

　　问:如何编辑Vista的多系统引导菜单BootManager?

　　答:MS在Vista中引入了BootManager来取代原有的Boot.ini管理多系统引导，编辑BootManager的内容需要使用BCDEDIT，详细内容可以查看搞定Vista5231多系统引导及使 用命令行BCDEDIT.EXE成功修改VISTA启动菜单!

　　注:由于近几个版本以来，Vista在Bootmanager上变化比较大，Vista和XP/2003还存在一些引导上的不同，使用该方法修改，可能还是无法彻底解决2级引导菜单的问题!
