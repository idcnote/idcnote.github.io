底部浮动条的一种兼容实现方法

内容摘要
这篇文章主要为大家详细介绍了底部浮动条的一种兼容实现方法，具有一定的参考价值，可以用来参考一下。
文章正文

这篇文章主要为大家详细介绍了底部浮动条的一种兼容实现方法，具有一定的参考价值，可以用来参考一下。

原理：在标准浏览器下用position: fixed的方式就可以了。
IE6下面用overlay.className = overlay.className迫使浏览器重新布局，以达到position: fixed的效果。

html代码如下：

  <style type='text/css'>
  html, body {
    margin: 0;
    padding: 0;
  }

    /* 这里不能加position: relative */
  }
  #content {
    font-family: 微软雅黑;
    font-size: 36px;
    line-height: 60px;
    width: 960px;
    margin: 0 auto;
  }
  #content p {
    text-indent: 2em;
  }
  #overlay {
    opacity: .5;
    filter: alpha(opacity=50);
    background: #ccc;
    width: 100%;
    height: 100px;
    position: fixed;
    _position: absolute;
    bottom: 0;
  }
  </style>
</head>
